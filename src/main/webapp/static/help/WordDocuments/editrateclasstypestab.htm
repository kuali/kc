Edit Rate Class Types Tab

 

Field

Definition

Rate Class Type

Required.  Enter the alphanumeric type of rate class this code is a part of, or select it from the lookup  or direct inquiry features.

Description

Required.  The actual familiar name or title of the rate class type that will correspond to the code value.

Prefix Activity Type

Required.  Check to add this rate class type as a prefix to the associated activity type, or clear to not prefix the activity type.  Activity Type is a major institution function.

Sort Id

A numeric value used to identify the way in which the rate class type is sorted (sequential order in which it is displayed in a list) in a database or user interface that displays data in a table format.

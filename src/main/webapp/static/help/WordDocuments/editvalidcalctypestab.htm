Edit Valid Calc Types Tab

 

 

Field

Definition

Calc Type Id

Required.  Uniquely-assigned identification number for this calculation type

Dependent Seq Number

Required.  The number used for the ordering of the calc type within sorted lists and data retrieval.  Alphanumeric.

Rate Class Type

Required.  Type the value if known in the box, otherwise, search for and select it to populate the box automatically by using either the lookup or direct inquiry tools.

Dependent Rate Class Type

The value that represents the rate class type that is dependent on this valid calc type

Rate Class Code

The value for the rate class that is to be associated with this valid calc type.

Rate Type Code

The value associated with the type of rate that the system applies to spcific cost elements (i.e. fringe benefits, F&A, inflation, etc)

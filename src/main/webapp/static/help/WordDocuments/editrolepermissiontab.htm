Edit Role-Permission Tab

The Role-Permission maintenance document contains one unique tab, the Edit Role-Permission tab.

 

Field

Description

Role Id

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions for the proposal development role.

Permission Id

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions for the permission you want to associate with the role.

Active

Select to make active, uncheck to deactivate.  Click within the checkbox  to select the item.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

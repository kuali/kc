Field

Description

Submission Details

Click the refresh button to display the current grants.gov submission status of the proposal, derived from the Grants.gov website.

Received Date

The date and time Grants.gov received the application

Last Modified Date

(Status Date)

The date and time the application status was updated last by Grants.Gov

Submission Status

The stage at which the proposal is currently at in the Grants.Gov process

Grants Tracking ID

The number assigned by Grant.gov upon successful submission of the proposal

Agency Tracking ID

 

A tracking number assigned by the agency, if applicable

Notes

Additional information regarding the status of the application.  For instance, this may include an explanation of the “rejected with errors” status.

After the opportunity has been selected, a remove opportunity button appears in the Opportunity Search section.  Additionally, a Submission History section appears.

Specify the Field Name Tab

The Specify the Field Name tab allows you to do just that – specify the field name – by selecting from the drop-down list (required) and indicating whether it is to be active or not.

 

Field

Definition

Field Name

Required.  Specify the name of the field by selecting one from the drop-down list.

Active

Select the check box if the field is to be active.  Clear the check box if it is inactive.

Document Roles (Permissions)

Document roles are user roles for particular document types within the KRA system.  They allow different types of access for different types of users for a particular document type.  This facilitates collaboration on the completion of an e-doc by multiple individuals, each of which take responsibility for portions of a document based on their area of expertise.

Routing Process

The Parameter document can be initiated by the members of the workgroup defined in the Parameter document and does not route for additional approvals if it is only modifying Parameter Values or changing the Parameter Constraint Code. New parameters or documents that modify additional fields need to be approved by the defined user or workgroup in the Parameter Maintainers Review route level.

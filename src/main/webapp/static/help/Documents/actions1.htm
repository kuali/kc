Actions

 

The Actions page of the Committee document is used to perform actions such as generating batch correspondence for members, viewing past correspondence, and printing portions of the document.

Figure 1472 Committee Document, Actions Page – Sections Layout

 

Table 618 Committee Document – Actions Page Section Descriptions

Section

Description

Batch Correspondence

Provides tools that allow you to generate batch correspondence by batch type and specify relevant dates to submit to the committee members.  Additionally, you are able to view previously-generated batch correspondence and history by batch type.

Print

Allows you to select portions of the committee document to be printed, and then print them.

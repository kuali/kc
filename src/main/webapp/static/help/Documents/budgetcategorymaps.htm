Budget Category Maps

Use the Budget Category Maps maintenance document to lookup, create, and update Budget Category Maps.  Budget Category Maps store the EDI and budget category requirements for each proposal submission form (EDI and Grants.gov) supported by Kuali Coeus.  Use the Budget Category Mapping e-doc (defined above) to map budget categories to maps created with the Budget Category Maps maintenance document.

To access the Budget Category Maps Lookup screen:

Maintenance main menu tab > Shared menu group > Budget Category Maps link

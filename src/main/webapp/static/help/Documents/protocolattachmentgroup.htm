Protocol Attachment Group

The Protocol Attachment Group maintenance document allows you to establish and maintain textual group names and associated numeric codes to which attachments that share a common purpose may be assigned.  Use this maintenance document to define groups that represent various types of file attachments that can be included with a Protocol e-doc (for example, those that are specific to personnel involved in the research).  As opposed to hard-coded values, these are paramaterized to facilitate customization by implementing institution.

Edit Protocol Attachment Group Section

The Edit Protocol Attachment Group section of the Protocol Attachment Group maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 651 Protocol Attachment Group Maintenance Document - Edit Protocol Attachment Group Section Field Descriptions

Field

Description

Code

Required.  Enter a numeric value that uniquely identifies the protocol attachment group table record in the system.  Click within the text box to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter the desired value in the box as necessary.

Description

Required.  Enter the full textual name of the protocol attachment group as it is commonly referred to.  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.  To work with larger screen real estate, click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.

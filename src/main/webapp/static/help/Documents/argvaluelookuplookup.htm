Arg Value Lookup Lookup

When you click the Argument Value Lookup link in the Shared menu group on the Maintenance main menu tab, KC displays the Arg Value Lookup Lookup screen:

 

Figure 454 Arg Value Lookup Lookup Screen

 

Use the Arg Value Lookup Lookup screen to search for and display existing Arg Value Lookup Ids.

Enter appropriate information in the search fields to find the Arg Value Lookup Ids you need (or leave all the fields blank to display all Arg Value Lookup Ids), then click  to display a list of Arg Value Lookup Ids.  When you click search, KC displays the Arg Value Lookup Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 455 Arg Value Lookup Lookup Screen with Search Information

 

KC displays the Arg Value Lookup Id that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy the Arg Value Lookup Id, Argument Name, Value or Description using the links in the Actions column.

For information about editing an existing maintenance document such as an Argument Value Lookup, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as an Argument Value Lookup, see Copying A Maintenance Document in Maintenance Document Operations.

When you institution creates a new Arg Value Lookup Id, it enters an Argument Name, Value, and Description for that id.

The Arg Value Lookup Lookup screen has a  button in the top right corner that lets you create a new Arg Value Lookup Id.

See the Argument Value Screen: Panels section below for additional information about creating a new Arg Value Lookup Id.

NASA Other Project Information V1.0

 

1.  Will NASA civil service personnel work on this project? This is hard coded to “N: No”.

1.a.  NASA FTEs information is intentionally left blank.

2.  Affect on historic sites is populated from the response to YNQ “Historical Sites Are Affected” (ID G6). 

2.a.  Explanation is populated from the Yes response Explanation of YNQ G6

 

3. International Participation is populated from YNQ “Project involves the International Cooperative activities?” (ID H1)

3.a.  If Yes, please mark all that apply:  Check all investigators and key persons (whose role is ‘Collaborator’).  If person is a KC

person (non rolodex), check country_of_citizenship from person table.  If this is null, default to USA.  

 

If person is non-employee person (rolodex person), check Organization from rolodex record to see if it is foreign (>=10). If there is no sponsor in rolodex record, check person’s country from rolodex address for non USA (not USA , PR, or VIR).

 

If after checking all people we have not found a foreign person, then Facility will be checked as default on form.

 

3.b: Explanation is populated from the Yes response Explanation of YNQ H1

 

 

4. Program Specific data: Upload Narrative Type NASA_OPI_ProgramSpecificData (ID 47). Only one of these types is allowed.

5. Appendices: Upload Narrative Type NASA_OPI_Appendices (ID 48). Multiple attachments of this narrative type are allowed.

6. Non-U.S. Organization Letters of Endorsement: Upload Narrative Type NASA_OPI_Non-U.SOrganizationLettersOfEndorsement" (ID 49).  Multiple attachments of this narrative type are allowed.

7. IRB & ACUC Letters: Upload Narrative Type NASA_OPI_IRB_ACUC_Letters (ID 50). Multiple attachments of this narrative type are allowed.

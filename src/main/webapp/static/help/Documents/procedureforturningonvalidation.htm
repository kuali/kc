Procedure for Turning On Validation

You turn on validation when you want to see errors and warnings for this proposal.  KC presents a list of validation errors and warnings with links to the specific data that you need to correct.  You can click the links to navigate directly to the data to correct it.  After correcting the data, you return to the validation screen to review any additional errors and warnings. 

 

1.  Click  to see the errors and warnings for this proposal.

 

KC displays the Validation Errors, Warnings, and Grants.gov Errors panels: 

 

Figure 193 Validation Panels

 

The Validation Errors panel contains errors that prevent you from submitting the proposal to KC workflow routing.  You can submit the proposal to KC workflow routing and print PDF forms when there are no errors listed in the Validation Errors panel.

The Warnings panel contains warnings that will not prevent you from submitting the proposal to KC workflow routing or www.grants.gov.

The Grants.gov Errors panel contains errors that prevent you from submitting the proposal to www.grants.gov.  You can submit the proposal to www.grants.gov and print PDF forms when there are no errors in the Validation Errors or Grants.gov Errors panels.

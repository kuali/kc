Negotiation Status

 

In the Negotiation e-doc, the KC system requires that the current status of the Negotiation e-doc is indicated via selection from a drop-down menu list.  That list of options available from which to select is controlled by this maintenance document.

The Negotiation Status maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including e-docs and other maintenance e-docs.

Default:  New negotiations are by default set to ‘In Process’ with this status code being set up as a parameter.  The default lookup table includes the following configurable selections:

•      In Process

•      Complete

•      Suspended

End Date Auto-Populate:  A related parameter controls which status code selections result in automatic population of the Negotiation End Date.

Edit Negotiation Status Section

The Edit Negotiation Status section of the Negotiation Status maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 991 Negotiation Status Maintenance Document - Edit Negotiation Status Section Field Descriptions

Field

Description

Status Code

The abbreviated and/or short version of the negotiation status.

Description

The full text version (long common name) of the negotiation status.

Active

Check to make active, uncheck to make inactive.

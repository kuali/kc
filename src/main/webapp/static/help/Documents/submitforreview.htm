Submit for Review

The Submit for Review action allows the investigators to submit protocols, amendments, renewals and response to revision requests to the IRB office. The system allows for investigators to optionally select the review type for the protocol and also the committee and schedule for the review of the protocol.

Configure Submission Review Type: To control which submission review type values should be visible to only the investigator and which should be visible to both investigators and the IRB Administrators, access Protocol Review Type from the Maintenance menu. For submission review type values such as (Full, Expedited etc.) that should not be visible to the investigator, set the Global Flag indicator to No. Submission Review Type values with  Global Flag value set to Yes will be visible to both investigators and IRB Administrators.

Configure Committee and Schedule selection: The system allows implementing institutions to control whether the investigators should be able to select the committee and schedule during protocol submission. This is enabled through the system parameter IRB_COMM_SELECTION_DURING_SUBMISSION accessible through the System Admin menu. The following options are available: M - Display committee and schedule and make their selection mandatory during protocol submission; O - Display committee an d schedule and make it optional to select committee and schedule during protocol submission.; D - Do not display or allow selection of committee and schedule during protocol submission.

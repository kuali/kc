XML Stylesheets

 

Click the XML Stylesheets link in the Workflow group on the System Admin screen to access the EdocLiteStyle Lookup screen, which allows you to search for, access, and export stylesheets and widgets used in the KC application.

Figure 2222EdocLiteStyle Lookup Search Result Example

Click export to have the stylesheet that you can then use in your e-doc application to ensure the same look & feel.

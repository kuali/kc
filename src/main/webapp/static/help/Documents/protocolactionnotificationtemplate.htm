Protocol Action Notification Template

 

The Protocol Action Notification Template is a non-standard maintenance screen that allows you to upload or replace templates used for notification of various actions taken by IRB Administrators or Researchers on Protocol documents that have been submitted for review.  The template file type is Extensible Stylesheet Language with the ‘.xsl’ extension.  These transform and render documents formatted in Extensible Markup Language (XML) to the Portable Document Format (PDF) output to make the content suitable for viewing online and printing.  This allows for the customization of the look and feel to mimic legacy paper correspondence templates at your institution.

Viewing

Click the view button in the Actions column for the desired Protocol Action Type row.

Depending on your browser and download manager type and settings, you may be prompted to select an application to open the file with (for example, Notepad).  Select the Open option, and click OK as necessary.  Otherwise, the application that is registered to open XML files by default will display the file.

Replacing

Prior to replacing a Protocol Action Notification Template, it is assumed you have started with the default, out-of-box template, made necessary modifications, and have validated the code (for example, using the Web Developer extension tools for the Mozilla Firefox browser:  https://addons.mozilla.org/en-US/firefox/addon/60/).

  To replace a protocol action notification template:

1.  Click the replace  button.  A Browse button appears in the File field:

2.  Click the Browse…  button.  For Windows users, a File Upload dialog box appears.  Use the Look in dropdown menu as necessary to locate and select the desired file, then click the Open button.  The dialog box closes and the File field displays the path and filename you selected.

3.  Click the upload  button.

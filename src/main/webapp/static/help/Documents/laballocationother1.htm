Table 694 Award Budget Document, Rates Page – Lab Allocation – Other Section Column Descriptions

Column

Description

Description

The type of rate that the system applies to specific cost elements.

On Campus

On or Off is displayed to indicate whether the rate applies to on-campus use or off-campus use.  The campus checkbox on Institute Rate is where this is set.

Fiscal Year

Identifies the institutional fiscal year for which a rate is effective.

Start Date

The start date of a budget period as reflected in the fiscal period in which the rates apply.  In the KC budget module, this is the date for which a rate becomes effective.

Institute Rate

Institutional rate associated with the rate type, on/off campus and fiscal year.

Applicable Rate

In the budget module this is the rate that is automatically applied to object codes within each budget version, where applicable.

 This rate may be different from Institute Rates for a number of reasons (for example, sponsor stipulations).

 

KC populates data in the “institute rate” and “applicable rate” fields with the current rates from the Unit Hierarchy rate table within the specific section in which the button appears.  (The sync rates button should only appear at the bottom of each section and not at the bottom of the Rates page).

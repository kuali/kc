Subaward Budgets

User may manually add line items as described below or use the Upload Sub Award option from the Budget Actions tab. to have the system generate the appropriate line items. If you have a system-to-system subaward file, use the Budget Actions > Subaward Upload feature to maintain these details and have the tool insert the line items into your budget. Please see the section on Subawards in the Budget Actions tab section.

1. Project Director

This is the PI info for the proposal. PI contact information comes from the Person Details panel of the ‘Key Personnel’ tab of KC proposal development.

 

2. Applicant Experience

Based on the YNQ [Question ID 26 in KC] from the ‘Questions’ tab within KC proposal development (from osp$eps_prop_ynq.answer) to set a value – “For Dept. of Education, Check "Yes" or "No" only if assistance is being requested under a program that gives special consideration to novice applicants. Otherwise, check "Not Applicable".

 

3. Human Subjects Research

Are any research activities involving human subjects planned at any time during the proposed project Period?  KC will check proposal’s special review to determine if human subject special review has been added. Only if special review type code is 1, this will set to Yes.  Are ALL the research activities proposed designated to be exempt from the regulations?  If special review approval status code is 4, set Exemption to “Yes”, and get exemption numbers from the special review Exempt # field.  If Exemption is “No”, we will get the Assurance number from the proposal‘s organization (osp$organizaton table).  Please attach an explanation Narrative

 

Please attach an explanation Narrative:

There is a proposal attachment type of “ED_SF424_Supplement_Attachment” in the ‘Abstracts and Attachments’ tab of KC proposal development [Narrative Type ID 54]. This attachment type can only attach one file.

Permissions

 

The Permissions page of the Protocol document allows you to specify users and assign them roles for preparing the Protocol document.  It consists of two tabbed sections:  Assigned Roles, which allows you to view assignees by role and click a button to view their rights; and Users, which allows you to select users, assign roles to them, and add this information as a part of the Protocol document.

Users with the ‘Modify All Protocols’ and ‘View All Protocols’ roles are not displayed on the Permissions page.

FIN_SYSTEM_INTEGRATION_ON Parameter

Another setup requirement for enabling  KFS Account auto-create from KC Award is the presence of a Parameter Maintenance Document in KC named FIN_SYSTEM_INTEGRATION_ON.  When the Parameter Value field is set to ‘ON,’ this indicates the financial system integration feature is active and enabled.  All fields and their values for this database parameter are shown in the screen shot below.

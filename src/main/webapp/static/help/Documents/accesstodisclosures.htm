Access to Disclosures

The system allows various users (reporters, administrators and reviewers) access to the reporter’s disclosures. Reporters are granted permission to access only their own disclosures. Reviewers are allowed to access disclosures assigned to them for review. Users with administrator privileges can access all of the reporter’s disclosures that fall under their jurisdiction based on the unit the administrator belongs to. In this section, the following user driven disclosure look-ups will be covered

Ø Disclosure look-ups for reporters

Ø Disclosure look-ups for reviewers

Ø Disclosure look-ups for administrators

 

Disclosure Lookups for reporters

1.   All My Disclosures: The reporter can access all of their own disclosures at any point of time through the “All My Disclosures” lookup on the researcher-landing page.

Ø To access this lookup navigate to Researcher  > Conflict of Interest   > All My Disclosures 

 

 >   >

 

                                Figure 459 Researcher/Conflict of Interest/ My Disclosures/ All My Disclosures

 

Ø Click on the All My Disclosures link. Various search parameters are provided to refine the search results. Only the signed-in reporter’s disclosures will be visible to the reporter. The Actions column in the search results provides an “open” action for the reporter to open the disclosure and take the appropriate action.

          Figure 460 Researcher/Conflict of Interest/ My Disclosures/ All My Disclosures > Search

                       

Disclosure Lookups for reviewers

1.   All My Disclosure Reviews: The reviewer can access all disclosures assigned to them for review through the “All My Disclosure Reviews” lookup on the researcher-landing page.

Ø To access this lookup navigate to Researcher  > Conflict of Interest > All My Disclosure Reviews

 >  >

 

         

          Figure 461 Researcher/Conflict of Interest/ My Disclosures/ All My Disclosure Reviews

Ø Click on the All My Disclosure Reviews link. Various search parameters are provided to refine the search results. Only disclosures assigned to the reviewed will be displayed. The Actions column in the search results provides an “open” action for the reviewer to open the disclosure and perform the review. Once the reviewer has completed her review and provided a recommended action, the system removes the disclosures from the reviewer’s queue.

6.   Undisclosed events: The administrator can access any of the reporter’s undisclosed events through the “Undisclosed events” look-up. Undisclosed events are system events such as IRB protocols, IACUC protocols, proposals, institute proposals and awards, which appear within the “New Project Disclosures to Complete” for the reporter.  The purpose of the “Undisclosed Events” look-up is to inform the administrator of events that need to be disclosed by reporters but have not been and to take appropriate action. Once the reporter has entered the data required to save a system event disclosure, the system event is no longer considered an “undisclosed event” and does not appear in the search results for “Undisclosed Events”

 

Ø To access this lookup navigate to Central Admin  > Pre-Submission Compliance > Undisclosed Events

  >  >       

 

         

          Figure 475 Central Admin/Pre-Submission Compliance/Undisclosed Events

Ø Click on the look-up icon to the right of undisclosed events to access the system events for which reporter has not provided a disclosure. The search results can be filtered by reporter or by event type among other search criteria.

 

Figure 476 Central Admin/Pre-Submission Compliance/Undisclosed Events > Search

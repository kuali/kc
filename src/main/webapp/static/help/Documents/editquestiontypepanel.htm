Edit Question Type Panel

The Edit Question Type panel lets you create a new Question Type Id.  You must enter an id (abbreviation) for the new Question Type Id and a Question Type Name.

 

Figure 626 Edit Question Type Panel

 

Table 3 Edit Question Type Panel Fields

Field

Definition

Question Type Id

Required

Enter a value to identify this Question Type Id.  You may use one or more letters and/or numbers.

Question Type Name

Required

Enter a descriptive name for this Question Type Id.

 

For information about creating a new maintenance document, see Initiating A Maintenance Document in Maintenance Document Operations.

For information about routing a maintenance document, see Submitting A Maintenance Document in Maintenance Document Operations.

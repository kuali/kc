Protocol Attachment Type Group

 

 

The Protocol Attachment Type Group maintenance document is used to establish and maintain numeric codes and values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs. 

This essentially associates groups of related Protocol Attachment Types (which themselves are numbered) together in a numbered group for efficient reference purposes.  The group is uniquely identified by this maintenance document, which itself is a combination of a type code and a group code.

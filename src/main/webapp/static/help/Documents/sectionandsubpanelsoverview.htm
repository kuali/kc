Section and Sub-Panels Overview

The Grants.gov tab in KC contains one Opportunity Search section and four sub-panels:

 

Table 38 Grants.gov Tab Section and Sub-Panels

Section/

Sub-Panel

Description

Opportunity Search

www.grants.gov opportunity look up feature; use this to search for a funding opportunity on www.grants.gov

Opportunity

Opportunity details, including the opportunity ID and title assigned by www.grants.gov and information on the type of application you submit

Submission Details

Submission details, including a tracking number and the submission date for the most recent submission of this proposal

Forms

Mandatory and optional federal forms you need to submit with this proposal using www.grants.gov

Submission History

Submission history for this proposal, including resubmissions with changes and/or corrections

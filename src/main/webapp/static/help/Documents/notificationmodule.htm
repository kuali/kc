Notification Module

 

The Notification Module (Notification Module Roles?) maintenance document allows you to use Roles and Qualifiers to establish system-generated notifications in KC.  Roles (for example, View Protocol) are associated with Modules (for example, IRB) and then together they are associated with Qualifiers (for example, Protocol Number) that specify particular data fields.  The Notification Module (Notification Module Role, Notification Module Roles) maintenance document is used to create a numeric record that associates a module, role, and qualifier combination that can then be available for selection in a Notification maintenance document.  The inherent permissions associated with roles, as they apply to certain data elements, work together to restrict possible action (such as the ability to view) and thus further refine recipent distribution lists.

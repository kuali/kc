Base Attribute

One of the required rule attributes for every Routing Rule. Base Attributes are comprised of:

•      Rule Template - the Rule Template used to create the Routing Rule.

•      Document Type - the Document Type to which the Routing Rule applies.

•      Active - a true/false flag to indicate if Routing Rule can be applied, i.e., if Routing Rule will be considered for a match.

•      From Date - the inclusive start date from which the Routing Rule will be considered for a match.

•      To Date - the inclusive end date to which the Routing Rule will be considered for a match.

•      Ignore Previous - a true/false flag to indicate if previous Routing for approval will be ignored.

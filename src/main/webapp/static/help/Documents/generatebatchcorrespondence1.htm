Generate Batch Correspondence

 

The Generate Batch Correspondence subsection of the Batch Correspondence is used to create the correspondence and submit it.  It allows you to specify correspondence by selecting a Batch Type and Protocol Expiration/Committee Action dates and submit the notifications to members.

Table 620 Committee Document, Actions Page – Generate Batch Correspondence Subsection Field Descriptions

Field

Description

Committee ID

Display-only based on entry made on the Committee page.

Batch Type

Required.  Select either Protocol Renewal Reminders or Reminder to IACUC Notifications from the list.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Committee Name

Display-only based on entry made on the Committee page.

Protocol Expiration / Committee Action

To enter or select the desired dates in the from and to fields, click the calendar icon to select, or type in mm/dd/yyyy format to populate the box with the desired date.

 

Template Requirement:  Implementing institutions will implement institution-specific, committee-specific templates for batch correspondence.  The Batch Type list displays those that are available.  They are added by using the Batch Correspondence maintenance document, which is accessible from the Compliance group on the Maintenance menu.

For more information about batch correspondence templates, see “Batch Correspondence” on page Error! Bookmark not defined. in Maintenance.

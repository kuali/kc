Specify the Field Name Tab

The Specify the Field Name tab allows you to select the field name (required) and indicate whether the field will be editable or not.

 

Figure 588  Specify the Field Name

 

Table 279  Specify the Field Name components

Field

Definition

Field Name

Required.  Specify the name of the field by selecting one from the drop-down list.

Active

Select the check box if the field is to be active.  Clear the check box if it is inactive. Active (checked) means the field will be editable in Proposal Development.

 

 

You may modify any of these existing values.

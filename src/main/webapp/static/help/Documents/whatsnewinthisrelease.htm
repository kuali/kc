What’s New In This Release?

Release version 4.0 of Kuali Coeus features both brand new functionality (including a new module) and enhancements/fixes to previously-released functionality.

KC 4.0 includes functional equivalency with Coeus, as well as integration with Kuali Rice (KR) release version 2.0.

KC 4.0 includes the following new functionality:

•      In Pre-Submission Compliance:  The new Confict of Interest (COI) module (including the Disclosure e-doc and Financial Entity maintenance)

•      In Pre-Award:  The new Negotiation e-doc

•      In Post-Award:  The new Subaward e-doc, and Award Report Tracking

KC 4.0 also includes enhancements to the IRB module, and minor fixes to bugs in Proposal, Award & Budget.

 

 

Significant effort has been made to ensure this documentation has been updated appropriately to reflect the functionality changes and additions in each release.  However, it has been prepared in advance of the official release, and therefore is subject to further modification. 

In the event portions of this documentation are incomplete at the time of publishing, every effort has been made to notify you within the corresponding topic heading with an appropriate alert to that effect, which serves as a placeholder for future editions.

For a complete, detailed listing of new and changed functionality in this release, see the official “Release Notes” on the kuali.org site (also a separate text file included in the software download package).

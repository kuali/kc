Closeout Type

The Closeout Type maintenance document is used to create and maintain a list of reasons for the closing out of subawards, which can be selected from within the Subaward e-doc (for example, ‘Audit Completed’).

Search Criteria Section

The search criteria section of the Closeout Type maintenance document displays two text entry fields that allow you to refine and narrow results that display in a table format below it.

 

Table 1002 Closeout Type Maintenance Document -  Field Descriptions

Field

Description

Closeout Type

An abbreviated representation of the type of closeout used for unique identification and ease of reference in the system (for example, a numeric value).

Description

The textual name of the type of closeout as commonly referred to, this full title describes the purpose of the type of closeout and is associated with the numeric code value.

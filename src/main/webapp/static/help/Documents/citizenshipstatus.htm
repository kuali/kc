Citizenship Status

 

The Citizenship Status Lookup is an administration maintenance screen that allows you to look up valid values for citizenship status, including the Code and Name.

Figure 1186 Citizenship Status Lookup Screen – Search Criteria Section Layout

 

Table 831 Citizenship Status Lookup Screen – Search Criteria Section Field Descriptions

Field

Description

Citizenship Status Code

The short code value that uniquely identifies the Citizenship Status record in the system, typically an abbreviated, uppercased version of the Name.

Citizenship Status Name

The full textual name of the Citizenship Status as it is commonly referred to.

Active Indicator

Indicates whether or not the Citizenship Status type record is active in the system and available for lookup, viewing and/or selection.  Select the radio button by clicking within a circle to place a dot within it to indicate your selection of the desired option.

 

For more information about general lookup screen functionality, see “Lookup” on page 67  in Selection, Entry & Action Tools.

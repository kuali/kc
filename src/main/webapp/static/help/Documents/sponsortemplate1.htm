Sponsor Template

 

 

The Sponsor Template maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.

Existing Value Modification:  OK to change.  Changing values will be reflected in all of the templates already defined.

Edit Sponsor Template Contacts Section

The Edit Sponsor Template Contacts section of the Sponsor Template maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.  These contacts are for common award maintenance tasks like financial and technical report submission.  When synchronized with an award, the Sponsor Template updates the award with the contact information.

Edit Sponsor Template Comments Section

The Edit Sponsor Template Comments section of the Sponsor Template maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.  These comments might include information about the interaction with a sponsor in providing assistance in managing the award.  When synchronized with an award, the Sponsor Template updates the award with the comment information providing consistency for awards with the same Sponsor Template.

Edit Sponsor Template Terms Section

The Edit Sponsor Template Terms section of the Sponsor Template maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.  A Sponsor Term is a rule, regulation, or requirement that applies to the award (for example, restrict the purchase of large items of equipment).

Disclosures

This section will display a table of approved disclosures submitted by the reporter. A “view” button is provided to open the selected item. The list of approved disclosures are organized chronologically with the most recent approved disclosure displayed at the top and the oldest approved disclosure displayed at the bottom.

Table 299 COI > COI Master Disclosure > Disclosure Page >Disclosures section field descriptions

Field

Description

Event

This column displays the type of disclosure. The values can be IRB protocol, Annual, Proposal, Update, Manual Award etc.

Disposition Status

This column displays the disposition status of the disclosure. Values can be In progress, Exempt, Project Level Approval, No Conflict Exists, No Further Action, Best Practices Memo, Disclosed Interests Eliminated, Disclosed Interests Reduced, Disclosed Interests Managed, Submitted for Review, Under Review by COI Reviewer, Under COI Office Review, Under COI Committee Review, Awaiting Additional Information, Potential Relationship, Relationship Identified, Reviewer Identified Conflict, Unresolved and Disclosed Interests Unmanageable

Approval Date

This column displays the date the disclosure was approved.

Project Name

This column displays the title of the project. Annual disclosures and Update Master disclosures have no title and the Project name column will be empty for these disclosures.

Project Number

This column displays any comments about the attachment that were provided when the attachment was added.

Disclosure Status

This column displays the disclosure status of the disclosure. The values that can be displayed in this column are In progress, Routed for Review, Approved and Disapproved.

Sequence

This column displays the sequence of the disclosure. The sequence number indicates the chronological order in which the disclosures were approved. The oldest approved disclosure will have a sequence number of 1. Subsequently for each approved disclosure, the sequence number increments by 1.

Notifications

This column provides hyperlinks to any notifications between reporter and the COI office for the particular disclosure.

Actions

This column contains a view  button that allows the user to open the disclosure and view additional details.

Module Names

Menu Link:  Module Names

Lookup Screen:  Coeus Module Lookup

Maintenance Document Label:  Kuali Coeus Module

Channel Title:  Module Names

The Module Names maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.

Edit Coeus Module Section

The Edit Coeus Module section of the Module Names maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 497 Module Names Maintenance Document - Edit Coeus Module Section Field Descriptions

Field

Description

Module Code

Required.  Enter a value to identify this Module Code.  You may use one or more letters and/or numbers.

Description

Required.  Enter a descriptive name for this Module Code.  Click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.

S2S Submission Type

Lookup screen label:  Submission Type Lookup

Edit Submission Types

The S2S Submission Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.

The type of grant application package/proposal submitted.  For example, types of submissions include: (1) Pre-application (2) Application (3) Changed/Corrected Application

Edit S2S Submission Type Section

The Edit S2S Submission Type section of the S2S Submission Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 563 S2S Submission Type Maintenance Document - Edit S2S Submission Type Section Field Descriptions

Field

Description

Submission Type

Enter a numeric value to uniquely represent the type of grant application package/proposal submitted.

Description

Enter the textual name of the type of grant application package/proposal submission as it is commonly referred to.  For example, types of submissions include: (1) Pre-application (2) Application (3) Changed/Corrected Application

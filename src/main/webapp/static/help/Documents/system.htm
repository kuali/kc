System

 

The System group on the System Admin menu contains links to system-related maintenance functionality such as management of daily system messages that are displayed on the user interface, institution-specific maintenance of extended attributes for Person records, and a method for accessing the Routing & Identity Management Document Type Hierarchy screen.

Messages Of The Day

Takes you to the Messages Of The Day Lookup screen, which allows you to create new or modify existing messages that appear in the header area of KC.

Person Extended Attributes

Takes you to the KcPersonExtendedAttributes Lookup screen, which allows you to create new or modify existing additional fields for the Person document, including degree and appointment information.

Document Configuration Hierarchy

Takes you to the Routing & Identity Management Document Type Hierarchy screen, where you are able to view hierarchical information by document type, view document configuration, search for or add routing rules by node.

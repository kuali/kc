The Print Sponsor Form Packages panel lists the proposal’s sponsor-specific forms and gives you options to print them.

 

Figure 198 Print Sponsor Form Packages Panel

 

You cannot print sponsor forms until you correct all errors on the Data Validation panel.

 

 

Table 62 Print Sponsor Form Packages Panel Definitions

Field

Description

Print Sponsor Form Packages

Read only

The name of the sponsor form package

Print Form Checkbox

Optional

The indication to print the form

Click  to select this form to print. 

Click  again to clear the selection so the form does not print. 

Actions

Click  to select all forms in the list for printing.

Click  to clear all forms from the selection list.

Click  to print the selected (check marked) forms.

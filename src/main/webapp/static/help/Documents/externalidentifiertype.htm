External Identifier Type

 

The External Identifier Type Lookup screen allows you to search for and view external identifier type names and codes.  This reference information is used to record additional means of identification for users outside of standard KC system user information.

Figure 1083 External Identifier Type Lookup Screen – Search Criteria Section & Result Table Examples

 

Table 757 External Identifier Type Lookup Screen Field Descriptions

Field

Description

External Identifier Type Code

A unique code that identifies this external identifier type.  The base data values are:

ERAC=Electronic Research Admin Commons User

HR =Human Resources ID

LICENSE =Driver's License

LOGON=Logon ID

RFID =RFID Implant

SSN =Social Security Number

TAX =Tax ID

External Identifier Type Name

A descriptive label for this external identifier type.

Display Sort Code

Display-only. A numeric value that determines the order in which external identifier types are displayed in the dropdown list.

Active Indicator

Display-only. Indicates whether this external identifier type is active.

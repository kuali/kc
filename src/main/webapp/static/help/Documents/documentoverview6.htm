Document Overview

 

The Document Overview section of the Protocol page contains one required field and two optional fields that allow for text entry of general information about the Protocol document.  An example is shown below, followed by a table that describes the purpose of each field.

Figure 1102 Protocol Document, Protocol Page, Document Overview Section Layout Example

 

Table 318 Protocol Document, Protocol Page, Document Overview Section Field Descriptions

Field

Description

Description

Required.  Enter a brief summary of the document.  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.

  The description will appear as the subject line in workflow actions and notifications.

Organization Document Number

Optionally type to enter the appropriate corresponding number that uniquely identifies the relevant document within the organization when applicable.

Explanation

Explain in greater detail the purpose of the protocol.  Click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.  After saved, click the green arrow  symbol to view full text in a separate browser window.

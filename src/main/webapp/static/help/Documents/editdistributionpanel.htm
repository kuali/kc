Edit Distribution Panel

The Edit Distribution panel lets you create a new Osp Distribution Code.  You must enter a code (abbreviation) for the new Osp Distribution Code and a description for it:

 

Figure 515 Edit Distribution Panel

 

Table 217  Edit Distribution Panel Fields

Field

Definition

Osp Distribution Code

Required

Enter a value to identify this Osp Distribution Code.  You may use one or more letters and/or numbers.

Description

Required

Enter a descriptive name for this Osp Distribution Code.

 

For information about creating a new maintenance document, see Initiating A Maintenance Document in Maintenance Document Operations.

For information about routing a maintenance document, see Submitting A Maintenance Document in Maintenance Document Operations.

Training Lookup

When you click the Training link in the Miscellaneous menu group box on the Maintenance main menu tab, KC displays the Training Lookup screen:

 

Figure 703 - Training Lookup

 

Use the Training Lookup screen to search for and display existing Training Codes.

Enter appropriate information in the search fields to find the Training Code you need (or leave all the fields blank to display all Training Codes), then click  to display a list of Training Codes.  When you click search, KC displays the Training Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 704 - Training Lookup with search information

 

KC displays the Training Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy the Training Code and Description using the links in the Actions column.

For information about modifying an existing maintenance document such as Training, see Editing A Maintenance Document in Common Maintenance Document Operations.

For information about copying a maintenance document such as Training, see Copying A Maintenance Document in Common Maintenance Document Operations.

Your institution enters a Description for each Training Code when creating the code.

The Training Lookup screen has a  button in the top right corner that lets you create a new Training Code.

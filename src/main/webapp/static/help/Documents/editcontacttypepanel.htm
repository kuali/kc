Edit Contact Type Panel

The Edit Contact Type panel lets you create a new Contact Type.  You must enter a code (abbreviation) for the new Contact Type and a Description for it.

 

Figure 493 Edit Contact Type Panel

 

Table 199 Edit Contact Type Panel Fields

Field

Definition

Contact Type Code

Required

Enter a value to identify this Contact Type.  You may use one or more letters and/or numbers.

Description

Required

Enter a descriptive name for this Contact Type.

 

 

You may modify any of these existing values. Changing existing values will be reflected in all the templates that are already defined.

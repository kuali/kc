Financial Entities

KC displays your list of financial entities by entity status, grouping together those that are either active or inactive into subsections.  Only the most recent (latest) version may be edited.

Click the view button to display a summary of information in a pop-up window.

To navigate between versions of the financial entity, click the [<prev] and [next>] buttons to the right of the Version # of ‘x’ display in the header area of the popup window.

Click the edit button to display the Financial Entity subsection within the Active Entities section.  The fields display current values/settings/content, yet are editable just as they are on the New Financial Entity page.

 

 

  Save Button:  After you’ve clicked the edit button, another result is that the screen now displays a save  button at the bottom of the screen that allows you to save changes you make in edit mode.

 

 /

Click the deactivate button to make this financial entity inactive in the system.

The system presents you with a confirmation prompt:

Click the yes button to confirm.  The entity will now appear in the Inactive Entities subsection with the activate button in the Actions column.

Deactivation arequires you to enter a comment or note regarding the reason for deactivating the financial entity.

 

  Only the most current version of an inactive entity can be made active.

Click the activate button to move the financial entity to the Active Entities subsection of the Financial Entities tabbed section of the My Financial Entities page.

Click the view history button to view a popup window that displays an Updated table that summarizes the number of updates, date/time info, and username for recent updates made to this financial entity.

The history of changes for each financial entity includes the version number, last updated date/time stamp (for each entity version), and the user name of the user who performed the last update action.

Click the refresh button at the bottom of the screen to refresh the list of financial entities after a financial entity is added, updated, or deactivated.

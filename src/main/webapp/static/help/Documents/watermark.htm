Watermark

The Watermark maintenance document is used to add or modify a list of watermarks that are applied on protocol attachments. Watermarks consist of verbiage that an implementing institution wants to stamp on protocol attachments. They typically indicate the status of the protocol (e.g. approved) along with other protocol information such as how long the protocol is valid for (e.g. expiration date of protocol). The watermarks can be different for different protocol statuses. Once watermarks are established and active, they appear on every page of a protocol attachment.

 

Currently watermarks only appear within protocol attachments that are in PDF format. In a future release the watermark functionality might be applied to other common protocol attachment formats such as MS Word, MS Excel, jpg etc.

Any time a protocol attachment has multiple versions within the system, the prior versions of the protocol attachment are stamped with a watermark text of “INVALID”. The current version of the protocol attachment is watermarked based on the watermark defined for the current status of the protocol.

Edit Watermark

The Edit action allowed on watermarks (such as watermark text of Approved, Expired etc.) within the IACUC Watermark maintenance document allows you to maintain each watermark and modify as needed. Some of the watermark data that can be modified are watermark text, positioning, stylizing, and type of watermark.

Watermark Text

Required.  Enter the full text that will be displayed as the watermark on each page of the protocol attachment. Watermark Active Status

Required. Indicate whether the watermark should be applied or not by using the Active indicator. Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Watermark Type

Required. Indicate whether the watermark to be applied is text or an image. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

 

Position

Indicate the position of the watermark within each page of the protocol attachment. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Alignment

Indicate the alignment of the watermark within each page of the protocol attachment. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Font Size

Indicate the font size of the watermark within each page of the protocol attachment. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Position Font Size

Indicate the size of the watermark in relation to the page using the position font size data element. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

 

Font Color

Indicate the color you want the watermark to appear in. Select from various color options. Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

 

Attachment Image

Click the Browse…button to look for, select, and upload the desired image you want to use as the watermark.  The field displays the path and filename you selected and the attachment is uploaded upon save.

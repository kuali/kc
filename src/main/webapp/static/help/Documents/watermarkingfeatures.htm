Watermarking Features

To access watermarks, go to the Kuali Coeus Maintenance tab and open the Watermark maintenance table under Shared.

 

The Kuali Coeus system provides watermarks for different protocol statuses in the out of the box application. Performing a search against the watermark maintenance table displays the watermarks for different protocol statuses. A numeric identifier that appears in the Status Code column represents the protocol status. The group name indicates whether the watermark is for the IRB or the IACUC module.

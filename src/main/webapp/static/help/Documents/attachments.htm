Attachments

Figure 232 Proposal Summary Page > Attachments sub-panel consists of three subsections: Proposal Attachments, Personnel Attachments, and Institutional Attachments

The Attachments sub-panel provides the user with access to the uploaded narrative attachment files.

•      The Proposal Attachments sub-section contains the proposal narrative files with the Attachment Type shown, any user-entered Description, the Status (complete or incomplete) and a [view] button to open the file in a separate browser window.

•      The Personnel Attachments sub-section provides access to the person-specific documents (biosketch, current & pending, etc.). To view the file in a separate browser window, click the [view] button.

•      The Institutional Attachments sub-section provides access to the documents that are used to meet internal business requirements. To view the file in a separate browser window, click the [view] button.

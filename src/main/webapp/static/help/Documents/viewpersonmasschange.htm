View Person Mass Change

 

The KC system retains changes made to the data store as a result of new person mass change submissions that have been approved.  Records of these changes and associated functional e-docs affected by the changes are made viewable on the View Person Mass Change page of the Person Mass Change document.

 

 

Feed To Kuali Financial System:  Your institutional accounting system can be updated via a feed that will apply changed records flagged in KC.

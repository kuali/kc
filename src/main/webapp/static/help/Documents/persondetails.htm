Table 71 Proposal Development Document > Key Personnel Page > Person Section > Person Details Subsection - Field Descriptions

Field

Description

Proposal Person Role Id

The description of the role the individual will perform for this proposal. This field identifies one of the following title/roles for each key person working on the project:  Principal Investigator, Co-Investigator, or Key Person.  Display-only.  Identifies the role in which this key person is assigned for the proposed project.

Full Name

A complete identification of this individual. First and last name of the user as recorded in the system database table.

Email Address

The electronic mail location for this individual.  The e-mail address of the person.

Primary Title

The main work title for this individual.  The person’s job title.

Home Unit

The name of the unit to which this individual is mainly assigned. The main unit in the Human Resources (HR) system associated with an individual's position or institutional appointment.

eRA Commons User Name

The identifier used by NIH of the individual for this role.  The unique username (from 6 to 20 characters) associated with a personal user account needed to log in to the National Institute of Health electronic Research Administration system for biomedical research proposal submissions to grantor sponsor agencies.  The unique identifier (User ID) given to each individual registered with the NIH eRA Commons.

Pager Number

The phone number used to send a page to the individual. The number of a paging device or beeper.

Office Location

The primary office location for the individual (e.g. the address for which the individual receives their mail.). The location of the work office.

Address Line 1

The first address line of the individual’s office location; used to send correspondence via the US mail. This is the physical location information for each performing site.  The information is maintained in the Rolodex.

Address Line 3

The third address line of the individual’s office location; used to send correspondence via the US mail. Additional line for address information

County

The county where the individual’s office is located. The county in which the person works.

Postal Code

  The zip code where the individual’s office is located. The zip code for the work address.

Percentage Effort

The relative amount of time this individual will devote to this proposal as compared to other activities.  (Note: this amount does not flow through to the credit split or budget areas.) The time/effort that the Investigators, and/or other personnel, are committing to the sponsored project.  This must be the proportion of time in relation to all institutional commitments and obligations. The total effort cannot exceed 100% for all sponsored projects and work-related duties. 

User Name

The system login identifier for this individual. The KC user ID.

Office Phone

The telephone number located in the work area for this individual. The primary office telephone number with area code first.

Directory Title

The work title used for this individual in the institution directory. The job title as recorded in the employee directory maintained by HR dept.

School

The name of the institution in which the individual for this role is employed. The school within the institution this person works.

Fax Number

The phone number used to send copies electronically via a facsimile machine.  The person’s fax number with area code first.

Mobile Phone Number

The cellular phone number of the individual for this role. The person’s cellular telephone number with area code first.

Secondary Office Location

A secondary office or laboratory location in which the individual may work. If there is a location where the person sometimes works, but not most of the time, it is displayed here.

Address Line 2

The second address line of the individual’s office location; used to send correspondence via the US mail. The section line of the official work address, physical or mailing.

City

The third address line used for locating physical location of the space where the proposed research will be performed (e.g. city where the individual’s office is located. The city in which the person works

State

The state where the individual’s office is located. The state where the institution is located.  Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.

Country Code

The code for the country where the individual’s office is located. The country where the institution is located.  Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.

Faculty

Indicates this person is a member of the academic staff. Checked indicates the person is a faculty member. Click within the checkbox  to select the item.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Ad Hoc Recipients

 

The Ad Hoc Recipients section of the Protocol Actions page allows you to create FYI, APPROVE, and ACKNOWLEDGE action requests and specify recipients (persons or groups) for them.  The recipients access the requests generated in their action lists.

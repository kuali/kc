Amount Info

The Amount Info page of the KC SubAward document has a single, tabbed section that utilizes standard line item functionality tools to allow you to specify information pertaining to key monetary amounts associated with the subaward (obligated and anticipated change amounts).  Maintenance of subaward amount changes and the dates those changes become effective can be recorded on this page.  Additionally, you are able to attach related document files related to dollar values and official correspondence regarding the subaward amount information.

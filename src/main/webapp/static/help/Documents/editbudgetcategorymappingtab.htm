Edit Budget Category Mapping Tab

The Budget Category Mapping tab has three required fields used to define and modify budget category mappings.

 

 

 

Figure 482  Edit Budget Category Mapping Tab

 

Table 185  Edit Budget Category Mapping Tab components

Field

Definition

Budget  Category Code

Required.  A numeric value that uniquely identifies the budget category. Use the lookup to return a value for this field.

Mapping Name

Required.  The name of the mapping used to map the budget category code to the target category code.

Target Category Code

Required.  The unique category code in KC that the budget  Category Code is being mapped to.

 

You should not modify or delete existing values without assessing the impact on electronic proposal submissions.

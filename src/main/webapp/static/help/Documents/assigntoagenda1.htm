Assign to Agenda

The submit action assigns the protocol to the agenda of the Committee/Schedule in which the IACUC committee will review the protocol.

 

Table 515 Protocol Document, Protocol Actions Page, Request an Action Section, Available Actions – Assign to Agenda Action - Action Attributes

  Action attributes

Description

Who can perform action

IACUC Administrators are allowed to perform this action.

Protocol state prior to action

Prior to the action being performed, the protocol must be in the following state:

 

The protocol must have been assigned to a committee and schedule.

The protocol status must be Submitted to IACUC.

The submission status must be Submitted to Committee.

 

Protocol state after action

After the action is performed

 

The protocol status does not change.

The submission status changes to In Agenda

Comments

To enter comments regarding the assignment of this Protocol document to the agenda, click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.  Click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.  After saving, click the green arrow  symbol to view full text in a separate browser window.

Action Date

Required.  By default, this is the current date, but you can change it as necessary to specify the date you want to this action to become effective.  Click the calendar icon to select, or type in mm/dd/yyyy format to populate the box with the desired date.

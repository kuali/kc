System Timeout

Figure 9 Session Expiration Message

 

The system may “time out” after a period of inactivity which could result in lost work and require logging in again.  Consult your System Administrator for specific duration and consequences.  The KC default is 60 minutes.  Upon a session expiration, document locks will be released.  Each time you click the save button (or navigate to a new tab), the timer is reset.

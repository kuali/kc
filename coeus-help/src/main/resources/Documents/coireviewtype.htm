COI Review Type

The COI Review Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including e-docs and other maintenance e-docs.

Edit COI Review Type Section

The Edit COI Review Type (Edit CoiReview Type) section of the COI Review Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 809 COI Review Type Maintenance Document - Edit COI Review Type Section Field Descriptions

Field

Description

COI Review Type Code

Required.  An abbreviated form of the Description field (for example, a numeric value) used to uniquely identify the record in the system. 

Description

Required.  The long form of the COI Review Type Code field (for example, a common textual name) that describes the purpose of the COI Review Type Code.  Click the expand text  icon to display a pop-up window with an expanded text entry area if you want more screen real estate to type in, and then click the continue button to close the window and return.  After text has been entered and saved, click the green arrow icon to read it in its entirety in the larger pop-up window, and then click the close button to close the window and return.

Negotiation Location

 

The Negotiation Location maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including e-docs and other maintenance e-docs.

Edit Negotiation Location Section

The Edit Negotiation Location section of the Negotiation Location maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 788 Negotiation Location Maintenance Document - Edit Negotiation Location Section Field Descriptions

Field

Description

Location Code

Required.  The abbreviated form of the negotiation location (for example, ‘S’ for Sponsor).  This may be an alphanumeric value.

Description

Required.  The long form (textual name as commonly referred to) of the negotiation location.  Not to be confused with longer sentence text that describes the purpose of the record, this describes the abbreviated form (code).

Active

Required.  Check to make actively available in the system, uncheck to make dormant.

Attached Items

 

The attached items section shows details of the attachments added to the protocol upon clicking the   button. In addition it has the following features that allow for sorting attachments, displaying attachment meta data performing view and update actions and tracking versions of the protocol attachments.

Sorting Protocol Attachments

The sort feature allows you to select the attachment type and sort by one of the following four fields - Attachment Type, Description, Last Updated By or the Last updated timestamp.  Upon selection of the sort by field, the system will automatically display the attachment as per the sorting criteria defined.

If you select an attachment type from the  provided for the sort, the system will sort and display only the attachment type selected, using the value selected in the Sort By field.

Display attachment details on attachment panel in collapsed state

The protocol attachment show/hide panel will display the attachment type, the attachment description, the last updated by person and timestamp.

Figure 1137 Protocol Document, Notes & Attachments Page, Attachment in collapsed state

View, Replace and Delete actions

Each protocol attachment panel once expanded to display all its details shall allow the user to perform view, replace and delete actions. The view will open the protocol attachment in a new window. The replace will allow the user to select another file to replace the existing protocol attachment. The delete action will trigger a warning message to the user asking to confirm the delete action and once confirmed will remove the protocol attachment permanently from the protocol. 

Figure 1138 Protocol Document, Notes & Attachments Page, Attached Items – View Replace and Delete actions


Display protocol attachment version information

Protocols are versioned by the system anytime a protocol is returned to the investigator and resubmitted to the IACUC office. If a protocol attachment is replaced during resubmission of the protocol, the system will create a version of the protocol attachment.  Within the Attached Items, the system will display for each version of the protocol attachment, the name of the user who last updated the attachment, the date the attachment was created, the date it was last modified on and a description for the attachment.

The system will allow only the current version of the protocol attachment to be opened through the Protocol Attachments section. To view older versions of the protocol attachment, go to Protocol Actions/Summary&History/Summary and navigate to the previous sequence of the protocol to view the older protocol attachment.

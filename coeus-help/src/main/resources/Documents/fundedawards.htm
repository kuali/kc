Funded Awards

The Funded Awards section of the Institutional Proposal Actions page displays information about any Award that maintained this IP record as the funding source. The panel displays the Award ID, Version, Proposal Version and Account ID. The action to link an IP record to an Award is initiated from within the Award Document. When an IP record in Pending status is linked as a funding source to an Award document, the IP record’s status is updated to Funded.

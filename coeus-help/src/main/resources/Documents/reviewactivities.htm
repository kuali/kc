Review Activities

The Review Activities section contains information regarding specific IP review activities and dates.  There is one subsection for each review activity added. 

•      You can create a new review activity using the New Activities section. 

•      You can create a copy of an existing review activity using the New Copy column next to the Original column of the activity you want to copy.

Table 77 Intellectual Property Review Document, Review Activities Section - Field Descriptions

Field

Description

New Activities

Activity Number

Required.  .  The identifier for the activity; defined by your institution.  Enter the Activity Number into the box.

IP Review Activity Type Code

Required.  The identifier for the type of IP review.  Your institution defines its own IP Review Activity Type Codes.  Some examples of IP Review Activity Type Codes are Meeting and Telephone Call.  Click the dropdown selection tool to select an IP Review Activity Type Code.

Activity Date

Click the calendar icon to select, or type in mm/dd/yyyy format to populate the box with the desired date.

Comments

Enter comments related to the activity.  Click within the text box (or press the tab key from a previous field) to reposition the cursor so that it is within the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.

Add

Click the add  button to add your selections/entries in the New Activities subsection to the New Copy subsection below.

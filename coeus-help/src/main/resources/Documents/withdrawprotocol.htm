Withdraw Protocol

This action allows the protocol to be withdrawn by investigators or the IRB Administrator after it has been submitted to the IRB office. This action can be initiated by the investigator or by the IRB Administrator. The protocol can be modified and resubmitted to the IRB.

 

Table 322 Protocol Document, Protocol Actions Page, Request an Action Section, Available Actions – Withdraw Protocol – Action Attributes

  Action attributes

Description

Who can perform action

This action can be initiated by investigators or the IRB Administrators

Protocol state prior to action

Prior to the action being performed, the protocol must be in the following state:

 

The protocol status must Submitted to IRB

The submission status must be in Submitted to Committee or Pending

Protocol state after action

After the action is performed

The protocol status changes to Withdrawn

The submission status changes to Withdrawn

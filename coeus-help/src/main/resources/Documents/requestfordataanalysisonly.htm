Request for Data Analysis Only

The Request for Data Analysis allows the investigator to indicate to the IRB office that interaction with human subjects and active data completion has been completed for the research project. However, the data collected is still being analyzed and the protocol needs to be kept open and active for that purpose. The investigator may choose to add an attachment along with a description while submitting this action request. Following the Request for Data Analysis action, the IRB Administrator can mark the protocol as Active for Data Analysis.

New Institutional Proposals for Disclosure

 

Figure 447 New Institutional Proposal Disclosure > Sub-panel details hidden

 

This section will list the Institutional Proposals to be disclosed. Expand the Institutional Proposals sub-panel by clicking the [>show]  toggle button.

 

Maintenance note:  The proposals in the list are determined by the Proposal Status codes maintained in the Config Parameter INSTITUTIONAL_PROPOSAL_DISCLOSE_STATUS_CODES

 

 

Figure 448 New Institutional Proposal Disclosure > Sub-panel details shown

 

The New Institutional Proposals for disclosure sub-panel displays the reporter’s Institutional Proposal List.  The identifying information includes Institutional Proposal Number, Institutional Proposal Name, Sponsor, Start Date, End Date, and Action.

 

Table 140 New Institutional Proposal Disclosure > Field label definitions

Field

Description

Institutional Proposal Number

The system-generated number assigned to the development proposal document when it became an institutionally submitted record.

Institutional Proposal Name

The title of the proposal as entered by the Aggregator in the Development Proposal.

Sponsor

The name of the sponsor applied to this proposal.

Start Date

The proposed start date for this project

End Date

The proposed end date for this project

Action

Click on “Report Coi” to begin the New Institutional Proposal (event) Disclosure.

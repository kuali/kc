Institutional Proposal

 

The Institutional Proposal (IP) document is the master record for the submitted funding proposal.  Administrators update the IP document with documents, updated Special Review details, or other details during the pre-award phase. 

KC automatically creates the Institutional Proposal document when a Proposal Development document status is updated to Submitted to Sponsor.  Administrators can manually create an Institutional Proposal document by promoting a Proposal Log document. 

When an Institutional Proposal is selected for funding by a sponsor, from an Award document, the administrator can link the Institutional Proposal document to the Award document as the funding proposal.  KC uses information from the IP record to populate the Award document (examples include title, sponsor, proposal persons).  Administrators then add and/or modify the provided information in the Award record. When an IP record in Pending status is linked as a funding source to an Award record, the IP record’s status is updated to Funded.

Unlinking an IP record used as a Funding Proposal in an Award record is performed from the IP record from the Institutional Proposal Actions Tab > Funded Awards panel.

Access

You can access the Institutional Proposal document one of these ways:

To access an existing Institutional Proposal document:

•      Researcher menu > Proposals menu group > Search Institutional Proposals link

•      Unit menu > Pre-Award menu group > Institutional Proposal

•      Central Admin menu > Pre-Award menu group > Institutional Proposal

•      Action List screen > Document Id column > Document Id link

•      Doc Search screen > Search Results table > Document Id column > link

When using a lookup method:

•      Enter your search criteria in the Institutional Proposal Lookup screen

•      Locate the existing document in the results section and then click ‘return result’ to open the document.

 

TIP: When you first open the IP document from the search window, it will be in READ ONLY mode. Click the (edit) button located at the bottom of the screen to make any additions or changes to the document.

 

To create a new Institutional Proposal document:

•      Unit menu > Pre-Award menu group > Institutional Proposal

•      Central Admin menu > Pre-Award menu group > Institutional Proposal

  Creating an Institutional Proposal from a Pending Proposal Log

 

You can create an Institutional Proposal document by promoting a pending Proposal Log document to a full Institutional Proposal record.  This is done when you do not have an electronic proposal routed from the Proposal Development module; this will create a record for internal reference and institutional reporting.

 

1.  Click Institutional Proposal  on the Pre-Award menu on the Kuali Coeus Portal menu:

Figure 143 Institutional Proposal Add Button on the Pre-Award Menu

KC displays the Proposal Log Lookup screen:

Figure 144 Proposal Log Lookup Screen

 

2.  Search and return the Proposal Log that you want to use to create this Institutional Proposal.  After you select the Proposal Log document, KC creates a new Institutional Proposal document.  KC populates the Institutional Proposal document with information from the Proposal Log document.

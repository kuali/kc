User entries are required to populate the forms as noted by yellow-highlighted, bold boxed cells in the tables.

Maintenance issue: Zip+4 in your Person Table addresses

If the zip code submitted is only 5 digits, not zip + 4, you may notice a red box highlighting these entries when the application is processed at NIH eCommons. This does not fail validation at either Grants.gov or eCommons at this time.

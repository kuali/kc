Special Review

 

The Special Review section of the Special Review page allows you to select a Type, Status and optionally, an Exemption number and Protocol number along with Application, Approval, and Expiration dates for one or more special reviews.

Figure 1386 Award Document, Payment, Special Review Page – Special Review Section Example

 

Table 562 Award Document, Payment, Special Review Page – Special Review Section Field Descriptions

Field

Description

Add

The line where you make your entries and selections and click the add button, which adds them as numbered line items below.

Type

Required.  The type indicates which special review, if any, is relevant to the document (proposal/award/irb).
Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.

Approval Status

Required.  The option indicates if the special review relevant to the document (proposal/award/irb) is pending approval, or already approved.

Click the down arrow  to display the list and click on an item in the list to highlight and select it to populate the box with your selection.

Protocol ID

The unique identifier assigned to the protocol submitted for regulatory review. Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions.

Application Date

The date the protocol was submitted to the regulatory committee.  Enter the date in mm/dd/yyyy format or select it using the date selector by clicking on the calendar icon .

Approval Date

The approval date is the date the protocol/request for permission was approved by the relevant institutional committee/person. 
Enter the date in mm/dd/yyyy format or select it using the date selector by clicking on the calendar icon .

Expiration Date

The date, after which, the existing protocol expires
Enter the date in mm/dd/yyyy format or select it using the date selector by clicking on the calendar icon .

Exemption #

One of six categories of exempt research specified by the Common Rule (45 CFR 46.101(b).   If the proposed project is exempt, as defined by the different institutional committees, an exemption number is generally issued, and is entered into this field.  Scroll the spin box to view the list items from which to select.

Comments

Type to enter internal notes relevant to the protocol or special review item.

Actions

After line items have been added, they may be removed by clicking the delete button.

Attachments

The Attachments subpanel is used to upload documents specific to the sub-recipient FDP agreement. The distributed values are:  Attachment 4A, Attachment 5, Attachment 6, Budget, Scope of Work, Subaward, and Transmittal Letter.

Attachment Type

Use the drop-down  menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Selections include: Attachment 4A, Attachment 5, Attachment 6, Budget, Scope of Work, Subaward, and Transmittal Letter.

 

Description

Required: user can enter any character in this text box (200 characters max)

File ID:

 Click browse to use your operating system’s Choose File or File Upload dialog window to locate and select the file you want to attach

 Add action

Click [add] after browsing and selecting a file to add as an attachment to save this row to the table below.

 Reset action

Click [reset] to deselect your option and return to the default.

Last Update

Read only: Populates with the date and time the row was added.

Updated By:

Read only: Populates with the name of the user that added the row.

 Delete action

Click [delete] on a maintained row to remove the report.

Maintained Attachment actions

 

View

Click the view button to open the file in a separate browser window.

Delete

Click the delete button to delete the subaward attachment row.

Replace

Click the replace button to exchange the uploaded attachment. Use the browse button to locate the replacement file and then click the add button.

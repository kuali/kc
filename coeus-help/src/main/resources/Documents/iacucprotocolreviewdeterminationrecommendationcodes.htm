IACUC Protocol Review Determination Recommendation Codes

The IACUC Protocol Review Determination Recommendation Codes maintenance document is used to add or modify a list of determination recommendation codes. This data serves as reference information that is made available for display and selection in the “Review Type Determination” dropdown field in the Online Review page displayed to the reviewers within the IACUC protocol. Note – Values cannot be edited, just deleted or added.

Note: The values available for adding to the IACUC Protocol Review Determination Recommendation Codes maintenance document depends on the values within the IACUC Protocol Review Type maintenance table.

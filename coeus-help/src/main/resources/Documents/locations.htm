Locations

 

The Locations section of the Identity menu group on the System Admin screen houses links to administration maintenance documents used for maintaining location information such as the various campuses at your institution, countries of the world, counties within your state, zip codes, and the states within the U.S.A.  All KC users can look up and view valid values for these documents, but only Technical Administrators and System Managers have the ability to edit, copy, and create new codes using these documents, which do not route for approval.

The following five links are covered in this topic:

Table 1149 System Admin Menu > Identity Group > Locations Section Link Descriptions

Link Name (linked to documentation topic)

Brief Description

Campus

Identifies the different fiscal and physical operating entities of an institution.

Country

Assigns specific identifying codes to country names.

County

Assigns specific identifying codes to county names.

Postal Code

Defines zip codes by city and state cross-references.

State

Defines U.S. Postal Service codes used to identify states.

Attachments:

 

The Attachments section within the Master aggregates and displays attachments added through various manual, events, annual and update master disclosures. The attachments appear within the “Attached Items” section.

 

This section shows details of the attachments added to the disclosure upon clicking the   button. In addition it allows sorting the attachments by Description, Last Updated By and Last Updated. This is a useful feature when there are multiple attachments.

Field

Description

Attachment Type

This column displays the type of attachment – the values can be Management Plan, PI Submission and Other and might be customized by an implementing institute.

Contact Name

This column lists the person to be contacted for the attached file.

Uploaded By

This column displays the user name that uploaded the attachment.

Posted Timestamp

This column displays the time and date the attachment was posted.

Comments

This column displays any comments about the attachment that were provided when the attachment was added.

File Name

This column displays the name of the file.

Project

This column displays the title of the project the attachment pertains to,

Email

This column lists the email address associated with the attachment, if it were provided when the attachment was added.

Phone

This column lists the phone number associated with the attachment, if it were provided when the attachment was added.

Description

This column displays the textual information about the attachment.

Financial Entity

This column displays the title of the financial entity the attachment pertains to.

COI Attachment Type

The COI Attachment Type maintenance document is used to establish and maintain different attachment type selections presented to reporters, reviewers, and COI Administrators in the Disclosure View > Notes &Attachments subpanel. Add specific use types as desired to meet your business needs

Edit COI Attachment Type Section

The Edit COI Attachment Type section of the COI Attachment Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 797 COI Attachment Type Maintenance Document - Edit COI Attachment Type Section Field Descriptions

Field

Description

COI Attachment Type Code

Required.  An abbreviated form of the Description field (for example, a numeric value) used to uniquely identify the record in the system.

Description

Required.  The long form of the COI Committee Role Type Code field (for example, a common textual name) that describes the purpose of the COI Committee Role Type Code.  Click the expand text  icon to display a pop-up window with an expanded text entry area if you want more screen real estate to type in, and then click the continue button to close the window and return.  After text has been entered and saved, click the green arrow icon to read it in its entirety in the larger pop-up window, and then click the close button to close the window and return.

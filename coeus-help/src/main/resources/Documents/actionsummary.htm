Action Summary

The Action Summary section of the Time and Money document provides you with an executive-level summary of transactions that affect the obligated amounts and dates for an award.  Award transactions that affect the funding obligation are displayed in read-only columns.

 

Figure 1427 Time And Money Document, Action Summary Section Example

 

Table 577 Time And Money Document – Action Summary Section Column Descriptions

Column

Description

Notice Date

Read only.  The date of the underlying award notice. 

KC displays the Project Start Date from the Award Hierarchy section as the Notice Date

Transaction Type Code

Read only.  The reason for adding or modifying award data. 

KC displays the Project End Date from the Award Hierarchy section

Obligation Start

Read only.  The effective date of the current funding period.

Obligation End Date

Read only.  The end date of the current funding period.

Obligated Cumulative

Read only.  The total amount authorized by the sponsor for spending on the award to date, not including recipient cost sharing.

Obligated Change

Read only.  The increase or decrease in the Obligated Cumulative amount due to this transaction.

IN PROGRESS - Document Overview

The Document Overview section is a standard piece of functionality in KC that is common to many e-docs. When displayed, the Description field is required to save the document. A ‘config’ type parameter pre-fills the field and hides the Document Overview panel in Proposal Development, relieving the aggregator from this requirement. The defaulted description consists of the title, proposal number, PI name, sponsor name, and due date.  To hide the Document Overview panel, change the System Admin > Parameter maintenance document “HIDE_AND_DEFAULT_PROP_DEV_DOC_DESC” config parameter to “Y”.

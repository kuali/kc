Budget Versions

The Budget Versions page of the Award document has three tabbed sections that allow you to view an overview of the budget for the award, create new budget versions, and view the personnel and non-personnel limits.

Editable Fields:  Some award budget-related fields are editable in multiple locations, which allows Award and Award Budget Administrators to make changes in these sections without the need to open the Award Budget document itself.

Section (linked to subtopic)

Brief Description

Budget Overview

View key information about the budget, including start/end dates, identification information, rate and cost information.

Budget Versions

View existing and create new versions of the budget for an award, view summary information, open detailed Award Budget document, or copy a version.

Budget Limits

View limits and obligated total amounts for personnel and non-personnel items, total costs.

 

The Budget Overview and Budget Versions sections are shown expanded by default.

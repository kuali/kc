Budget Versions

 

 

The Budget Versions tab in the Award Budget assigns an incremental budget version to each Award Budget document that is added by the aggregator. Award budgets can be submitted, approved and posted to the financial system from KC (when KC is linked to a financial system).  When the budget is initially created, a budget version exists automatically, but if a budget is rejected or cancelled, the number assigned to it can be used for the next version.

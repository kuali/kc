Batch Correspondence History

 

The Batch Correspondence History subsection of the Batch Correspondence section allows you to filter the view of previously-submitted batch correspondence by Batch Type according to a range of Run Dates you specify.

Figure 1072 Committee Document, Actions Page – Batch Correspondence History Subsection Example

Table 365 Committee Document, Actions Page – Batch Correspondence History Subsection Field Descriptions

Field

Description

Committee ID

Display-only based on entry made on the Committee page.

Batch Type

Required.  Select either Protocol Renewal Reminders or Reminder to IRB Notifications from the list.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Committee Name

Display-only based on entry made on the Committee page.

Run Date

Select a range by entering or selecting a date in both the from and to fields.  Click the calendar icon to select, or type in mm/dd/yyyy format to populate the box with the desired date.

Click the filter button to cause subsections to appear below for each type by date range.

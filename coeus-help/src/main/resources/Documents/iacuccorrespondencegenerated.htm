IACUC Correspondence Generated

 

The IACUC Correspondence Generated maintenance document allows you to select a combination of a IACUC Protocol Action Type and a IACUC Correspondence Type that together become a single “correspondence generated” record.  It is used to establish and maintain numeric code values that are stored in database tables.  This data serves as reference information that is then made available for display and selection in various areas of the system, including other e-docs.

Edit IACUC Correspondence Generated for Protocol Action Section

The Edit IACUC Correspondence Generated allows you to specify and maintain the details associated with the reference table record in fields.

Table 912 Edit IACUC Correspondence Generated

Field

Description

IACUC Protocol Action Type

Required.  Select an option from the list as desired.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

IACUC Protocol Correspondence Type

Required.  Select an option from the list as desired.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Final Flag

Click within the checkbox to select the item to indicate the IACUC Correspondence Generated record is final.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.  When selected, the displayed value in the saved document appears as ‘Yes’.

Template

The information provided in the Template subpanel is used to populate the FDP agreement forms.

Figure 1536 Subaward > Template Information> Template panel - new document

 

Field Name/Title

Required?

Description

SOW/Budget specified in Sub Proposal

No

The Yes or No radio button choice populates the FDP Agreement form T&C 1 as follows:

Yes – box checked for ‘As specified in Sub recipient’s proposal’

No – box NOT checked for  ‘As specified in Sub recipient’s proposal’ (Default of ‘No’)

Sub Proposal Date

No

Enter or use the calendar icon tool to select a date (past, present or future date) to populate the FDP Agreement form T&C 1 ‘proposal dated’ field..

Invoice/Payment Contact

No

Use the drop-down list to select a contact; populates in FDP Agreement form T&C 2. Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

Final Statement of Costs Contact

No

Use the drop-down list to select a contact; populates in FDP Agreement form T&C 3.

Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

Change Requests Contact

No

Use the drop-down list to select a contact; populates in FDP Agreement form T&C 6.

Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

Termination Contact

No

Use the drop-down list to select a contact; populates in FDP Agreement form T&C 8.

Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

No Cost Extension Contact

No

Use the drop-down list to select a contact; populates in FDP Agreement form T&C 9.

Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

Performance Site same as Org Address?

No

The Yes or No radio button choice populates on the FDP Agreement form Attachment 3B (Sub recipient Contacts) with the response for the question ‘Is the Performance Site the Same Address as Above?’

Performance Site same as PI Address?

No

The Yes or No radio button choice populates on the FDP Agreement form Attachment 3B (Sub recipient Contacts) with the response for the question  ‘Is the Performance Site the Same Address as the PI address below?’

Sub Registered in CCR?

No

The Yes or No radio button choice populates on the FDP Agreement form Attachment 3B (Sub recipient Contacts) with ‘the response for the question ‘Sub recipient currently registered in CCR?

Parent Congressional District (if applicable)

No

Text entry field (20 characters maximum) will populate the (Parent) ‘Congressional District’ field in FDP Attachment 3B.

Parent DUNS No. (if applicable)

No

Text entry field (20 characters maximum) will populate the (Parent) ‘DUNS NO.’ field in FDP Attachment 3B.

Copyrights/Data Rights

No

Use the drop-down list to select an option; populates in FDP Agency-Specific Attachment 2 forms ‘Special Terms & Conditions section Copyrights’ field.

Exempt from reporting exec compensation

No

The Yes or No radio button choice populates on the FDP Attachment 3B and 3B page 2 with the response for the question ‘Is Sub recipient exempt from reporting compensation?’

Carry Forward Request Sent to:

If Automatic Cary Forward answer = Yes

Use the drop-down list to select an option;  populates the FDP Attachment 2 forms ‘Special Terms & Conditions’ section ‘If No, Carry forward requests must be sent to Prime Recipient’s’ field.

Options available in this drop-down list are maintained in the Contact Type table (those associated with the Subaward module in the Contact Usage table).

Automatic Carry Forward

No

The Yes or No radio button choice populates the ‘Automatic Carry Forward’ field in the FDP Agency Specific Attachments 2 forms ‘Special Terms & Conditions’ section ‘Automatic Carry Forward’ field.

Yes populates ‘Yes’ and omits ‘If No, Carry forward requests must be sent to Prime Recipient’s’ text.

No populates ‘No’ and includes ‘If No, Carry forward requests must be sent to Prime Recipient’s’ text

Applicable Program Regulations

No

Text entry field (50 characters maximum) will populate the FDP DOE-Specific Attachment 2 form ‘General Terms & Conditions’ section ‘Applicable Program Regulations’ item 2.

Treatment Program Income Additive

No

The Yes or No radio button choice populates the FDP NIH-Specific Attachment 2 form ‘General Terms & Conditions’ section ‘Treatment of Program Income’ item 6.

Yes populates ‘Additive’ and omits the ‘Other, Prime Recipient specify alternative from NIH Agreement’ text.

No omits the ‘Additive’ text and populates ‘Other, Prime Recipient specify alternative from NIH Agreement’ text.

Applicable Program Regulations Date

If ‘Applicable Program Regulations’ = ‘Yes.’

Enter or use the calendar icon to select a date (past, present or future date) to populate the FDP DOE-Specific Attachment 2 form ‘General Terms & Conditions’ section ‘Applicable Program Regulations’ date for item 2.

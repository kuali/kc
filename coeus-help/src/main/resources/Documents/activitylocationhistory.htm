Activity/Location History

This exists so that you can select an option to view Activity/Location History containing all Negotiation Activities associated with the Negotiation record, including key identifying information for each activity and/or location.

Figure 224  Negotiation Document > Activities & Attachments Section > Activity/Location History Subsection – Example

The Activity/Location History subsection provides you with the ability to optionally export this report as either a CSV, spreadsheet or XML file.

For more information, including complete instructions on how to use these features, see “Export Functionality” on page 59 in Overview > Selection, Entry & Action Tools.

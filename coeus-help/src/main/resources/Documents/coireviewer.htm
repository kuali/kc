COI Reviewer

COI Reviewers evaluate and recommend approval or disapproval of internal Disclosure documents after reviewing them, and also update the status in the process.  Different types of COI Reviewers (for example, Committee members or COI Administrators) and different types of reviews (for example, FYI or Acknowledge) dictate the permissions associated with the Disclosure e-doc (for example, the ability to add attachments or return to the Reporter).

Edit COI Reviewer Section

The Edit COI Reviewer section of the COI Reviewer maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 928 COI Reviewer Maintenance Document - Edit COI Reviewer Section Field Descriptions

Field

Description

Reviewer Code

Required.  An abbreviated form of the Description field (for example, a numeric value) used to uniquely identify the record in the system. 

Description

Required.  The long form of the Reviewer Code field (for example, a common textual name) that describes the purpose of the COI Reviewer.  Click the expand text  icon to display a pop-up window with an expanded text entry area if you want more screen real estate to type in, and then click the continue button to close the window and return.  After text has been entered and saved, click the green arrow icon to read it in its entirety in the larger pop-up window, and then click the close button to close the window and return.

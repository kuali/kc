Route Log

A special workflow function that displays the Routing history of a given document or selected documents. The documents are displayed in a separate window from the action lists, and Documents Searches. The Route Log displays a table of general document information for the document and a detailed list of current Actions Taken and pending Action Requests.  The Route Log is a web interface that displays information about the Routing of the selected documents. The Route Log is typically accessed from either the Action List or Document Search. It displays a table of general document information for the document and a detailed list of current Actions Taken and pending Action Requests. Can also be considered as an audit trail for a document.

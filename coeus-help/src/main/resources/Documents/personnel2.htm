Personnel

 

The Personnel page of the Protocol document is where you specify the individuals involved in the Protocol. 

Personnel Changes & Protocol Status:  The document status displayed in the header area of the document dictates changes to personnel information.  For example, in Pending/in Progress status, no amendment is necessary.

Figure 1092 Protocol Document - Personnel Page Layout

The Personnel page has three main component areas that allow for the selection, entry and maintenance of person information.

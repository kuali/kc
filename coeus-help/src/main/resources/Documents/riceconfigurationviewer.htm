Rice Configuration Viewer

 

The Configuration Viewer workflow screen allows you to view Configured Properties for Kuali Rice modules.  A table displays content in two sortable columns:  Config Key and Config Value.

Configured Properties:

Config Key

The name of the configuration key.  Click the column header link to sort the contents alphabetically (ascending/descending toggle).  Keys typically include periods, primarily lowercase letters, with no spaces.

Config Value

The value of the configuration key.  Values are alphanumeric and may include (for example, directory paths, URLs, true/false, yes/no).

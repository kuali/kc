Entity Data Groups

The Entity Data Groups maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including e-docs and other maintenance e-docs.

Figure 1860 Entity Data Groups Maintenance Document -  Layout

Edit Entity Data Groups Section

The Edit Entity Data Groups section (Fin Entities Data Groups) of the Entity Data Groups maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 819 Entity Data Groups Maintenance Document - Edit Entity Data Groups Section Field Descriptions

Field

Description

Data Group Id

Required.  An abbreviated form of unique identifier (for example, a numeric value) that is associated with the textual name in the Name field, this is used for efficient reference and organization in the system.

Data Group Name

Required.  The textual name as commonly referred to (for example, ‘Remunerative Activities’).

Data Group Sort Id

Required.  This is a numeric value that controls the sequential order in which the relationship type appears in a list.

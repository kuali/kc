Modular Budget

The Modular Budget page of the Award Budget document contains a selection section and three expandable sections.

Figure 1679  Award Budget Document > Modular Budget Page – Sections Layout

 

Table 724 Award Budget Document > Modular Budget Page – Section Descriptions

Section (linked to subtopic)

Brief Description

Select Modular Budget Period

Select a budget period from a dropdown list and click a button to update the overview and cost sections to display data from this period.

Modular Budget Overview

View start/end dates and total project requested cost.

Direct Cost

View direct cost less consortium F&A, consortium F&A, and total direct cost.

F&A

View total funds requested, F&A rate & base by type.

 

  The sync button synchronizes the detailed budget to the modular budget.

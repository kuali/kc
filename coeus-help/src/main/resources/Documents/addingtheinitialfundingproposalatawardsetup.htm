  To add a Funding Proposal to an Award:

•      Click the lookup  icon to find and select an Institutional Proposal Number on the Lookup screen:

 

 

Figure 1283 Proposal Lookup Screen

•      Enter appropriate information in the search fields to find the Proposal Number you need, or leave all the fields blank to display all Proposal Numbers, and then click  to display a list of Proposal Numbers.  When you click , KC displays the results of your search below the search button:

Figure 1284 Proposal Lookup Results Information

KC displays the Institutional Proposal Numbers that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

•      In the search results list, click return value for the Proposal you need to add to this Award.

KC displays the KC Award screen again with the selected Institutional Proposal Number in the Add Funding Proposals section:

Figure 1285 KC Award, Add Funding Proposal Section with Proposal Number Entered from Search Results

When you click in the Add Funding Proposals section, KC displays this Funding Proposal in the Current Funding Proposals section of the Funding Proposals section.  KC also provides a read-only Details subsection of the selected IP.

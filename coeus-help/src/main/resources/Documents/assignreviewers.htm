Assign Reviewers

The Assign Reviewers action retrieves active members of the IRB committee selected and displays them for the administrator to make primary/secondary reviewer assignments. For full board review protocols, the Assign Reviewers action becomes available after committee and schedule are selected. For expedited review, exempt protocols and notify IRB (FYI) submissions, the Assign Reviewers becomes available after the committee is selected. The system does not require schedule to be selected prior to making “Assign Reviewers” available for expedited, exempt and FYI submissions.

 

Table 246 Protocol Document, Protocol Actions Page, Request an Action Section, Assign Reviewer Action – Action Attributes

  Action attributes

Description

Who can perform action

IRB Administrators are allowed to perform this action.

Protocol state prior to action

Prior to the action, the protocol must be in the following state:

 

The protocol must have been assigned to a committee. Schedule assignment is needed for full committee submissions.

The protocol status must be Submitted to IRB.

The submission status must be Submitted to Committee or In agenda.

 

Protocol state after action

After the action is performed the protocol status and submission status does not change.

COI Note Type

The COI Note Type maintenance document is used to establish and maintain different types of notes that can be added to a COI disclosure by a reporter, COI administrator or COI reviewer.  These note types are available for selection within the Note Type selection list in the Notes and Attachments panel of the disclosure.

Edit COI Note Type Section

The Edit COI Note Type section of the COI Disclosure Event Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 924 COI Note Type Maintenance Document - Edit COI Disclosure Event Type Section Field Descriptions

Field

Description

COI Note Type Code

Required.  This is numeric identifier that is unique to each note type

Sort Id

Required. This field requires a numeric identifier and controls the order in which the notes will appear in the disclosure.

Is Active

Required. Check this field to indicate the note type is active. Uncheck to indicate the note type is inactive. Only active note types will appear in the selection list for the note type within the disclosure.

Note Type Description

Required.  Enter the note type description here. This is the value that will appear in the note type selection within the disclosure.

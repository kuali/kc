NEW TOOL: User Attached Forms

KRACOEUS-6855 released in 5.2.1

This tool enables users to upload individual “unstitched” Adobe forms and/or any Optional forms “stitched” into a regular Adobe Opportunity to meet submission requirements when the target form(s) is not mapped for Coeus submission to Grants.gov. This solution means that the data is not entered into KC fields; these filled forms are treated more like uploaded narrative files and are submitted with the s2s opportunity to Grants.gov.

 

Things to know about using the attachment tool:

Where do I find the forms?

Unstitched forms are provided by Grants.gov. They can be retrieved at the Grants.gov web page. http://www.grants.gov/web/grants/forms/r-r-family.html#sortby=1 (or navigate to grants.gov and select “Forms.”

 

Forms listed with an asterisk (*) in the PDF column are unstitched and available for use with the upload tool.

Note: the comment at the top of the table – that the forms are “not submittable” – means that they are not intended to be submitted individually to Grants.gov. These special forms have been ‘unstitched’ to be included in our S2S submissions as part of a full submission that includes the other forms in the posted opportunity.

 

Figure 2167: Grants.gov Forms page listing 'unstitched' forms

What about the posted Adobe Opportunities? Can I use those?

Yes, but you can only use the Optional forms in a packaged opportunity. If the form you require is listed in the Mandatory section, you will need to retrieve it as an individual unstitched form from the Grants.gov web site. Mandatory forms cannot be processed in the upload translation.

Figure 2168: Example of a traditional Adobe Opportunity with an Optional form Complete.

Can I upload the extracted Subaward Budget Grants.gov forms here too?

No. You must upload the subaward budget files in the Budget module where the financial data can be extracted and inserted into your detailed budget.

You should only be using this tool to upload the forms that are unavailable in KC because they have not been programmed or mapped to KC data.

 

How do I fill out the form?

Unstitched forms do not have a validation tool on the page. If you are using an unstitched form, you may wish to also download your submission’s Grants.gov opportunity to use as a reference. If you upload an incomplete unstitched form to the User Attached Forms tool, it will not translate, which means it did not validate and cannot be included in the Grants.gov submission until it validates. You will receive the grants.gov form schema –level error message. (The error message may not provide specific faults, but will confirm that the form is incomplete and does not meet the validation requirement.) You must update the form and repeat the upload.

Adobe Opportunity Packages: If you want to use an Optional form in an Adobe Opportunity, click in the box beside the form name to select it for inclusion. You may then click on the form name to navigate directly to it, or scroll down until you get to the form.  Follow sponsor-specific instructions for your submission to complete the data requirements. The minimum form-level data requirements can be verified by using the “Check Package for Errors” button on the cover page. I you receive the “Complete” mark; the form has met the minimum validation. (This may not meet the sponsor’s requirement.) You must provide at least the minimum requirement to successfully translate the form.

The tool does not have the ability to support Mandatory forms in an Adobe Opportunity, due to the “stitched” nature of data flow between forms. If the form is mandatory, you will need to locate and use the unstitched form from Grants.gov.

 

When should I connect to the Grants.gov Opportunity?

v As with any S2S connection to Grants.gov opportunities, if the all the mandatory forms are available, you can connect to the opportunity at any time.

v If the forms you will be uploading using the User Attached Forms tool are Optional, you can connect to the opportunity either before or after you upload the form:

o  If you connect before uploading the forms with the tool, the Grants.gov Forms section will show those optional forms as “unavailable”.

o  If you connect to the Grants.gov opportunity after uploading the forms to your proposal, when you complete the connection to the opportunity, the Grants.gov Forms Description column will note ‘User Attached Form” in those form rows and allow you click the box to include that optional form in the submission.

v If there are mandatory forms in the opportunity that are not available in KC, you will need to upload those unstitched forms first and then connect to the opportunity. We have added the names of the unavailable forms to the ‘not eligible for KC transmission to Grants.gov” message window to alert users to the specific forms making the connection ineligible and the need to use the new tool to meet the requirement.

Notice at the bottom of the message, the Missing Mandatory Forms List has been added to inform the user of the specific missing forms.

 

You must SAVE your proposal document after you upload the User Attached Form file(s) before you use the s2s Lookup and connect to an opportunity!

•    If you don’t save your document prior to the connection, the uploaded s2s forms will not be recognized by the system and remain described as unavailable.

•    If this happens, delete the uploaded s2s form and re-upload.

•    A resolution (alert, contextual message on screen, alteration to the upload/save process) will be provided in a forthcoming release.

 

 

Using the User Attached Form Upload tool:

New panel on the Proposal S2S tab > User Attached S2s Forms

Expand the User Attached Forms subpanel to use the S2S User Attached Form upload tool.

 

•      Navigate to the S2S tab and expand the  User Attached Forms subpanel

•      In the Add row, enter some information about this upload in the Description field.  Ex. the name of the target form, like “Planned Report” if you are uploading a single unstitched form. (This field is not used in the transmission.)

•      Click Browse to open a search tool window; locate your unstitched form or Adobe opportunity with the target form completed. When you complete the search, the File Name field will display the location path of your upload,

•      Click the [add] button. If the target form(s) are complete, the Namespace field will update with the official form information, and the Form Name field will update from the form type name; the Form File Name will appear (name the user gave the file).

•      SAVE your proposal document by clicking the (save) button at the bottom of the screen.

You must SAVE your proposal document after you upload the User Attached Form file(s) before you use the s2s Lookup and connect to an opportunity!

•    If you don’t save your document prior to the connection, the uploaded s2s forms will not be recognized by the system and remain described as unavailable.

•    If this happens, delete the uploaded s2s form and re-upload.

•    A resolution (alert, contextual message on screen, alteration to the upload/save process) will be provided in a forthcoming release.

 

 

Actions:

Once a user attached s2s form has been successfully uploaded, the following actions are available to the user:

 

Fields:

Definition

No (number)

Line number – to identify individual items. Will be filled in by the system

Description

User entered textual information about this item.

Namespace

System-entered URL identifier of the upload s2s form.

Form Name

System entered – the Grants.gov form name of the uploaded, translated file.

Form File Name

The name the file was given by the user when the GG form or opportunity was downloaded and saved.

 

Buttons:

Definition

Browse…

Click “browse” to open a search window. Locate the filled Adobe form and select it to close the window. The selected upload will appear in the “Form File Name” field.

add

Click “add” to complete the upload and add a row to the table. You must enter a description and browse and select a valid form prior to using the add button.

View XML

Click to open a separate browser window and view the XML data.

View PDF

Click to open the PDF form in a separate browser window. If you have uploaded an Adobe Package, the entire package will generate.

Delete

Use the “Delete” button to remove all the contents of this row.

 

Viewing and Printing the User Attached s2s Forms:

At 5.2.1 release (May 2014), most of the unstitched forms can only be print/previewed via the “View PDF” button in the S2S User Attached Form subpanel.  The PHS Cumulative Inclusion Report 1.0 and the Planned Report 1.0 can be print-previewed with the Forms > Print Selected option because we have the NIH provided style sheets for these forms.

 

To Print: use the View PDF button in the Actions column to generate a PDF file that can be printed, if desired.

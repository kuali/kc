RR Personal Data V1.2

The Grants.gov Grants.gov Research & Related (RR) Family form Research & Related Personal Data Form V1-1 & V1-2, prints information for all investigators indicated in the Key Personnel screen of KC Proposal Development. The print order of the investigators depends on the sort order in the Key Personnel screen.

 

 

Name fields

Only First name and Last name are currently being set based on PI or Co-PI information on the Key Personnel tab.

 

Gender

This information is not mandatory therefore it is not populated or maintained currently by KC.

 

Race, Ethnicity, Disability Status, Citizenship

This information is not mandatory therefore it is not populated or maintained currently by KC.

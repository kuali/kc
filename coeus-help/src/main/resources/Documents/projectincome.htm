Project Income

Project income is gross income earned by the University that is directly generated by a sponsored activity, or earned as a result of an award.  In cases where income is anticipated, it should be reported to the sponsor.  Sometimes, the sponsor will reduce the award based on the anticipated project income. At other times, the project income will be used as additional support for the project.  (The exact disposition of these funds is usually stipulated in the sponsor’s program announcement and/or award guidelines.)

The Project Income screen allows you to detail how much income is anticipated by the project by period, as well as the source of the income.

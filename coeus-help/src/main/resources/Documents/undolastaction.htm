Undo Last Action

This action allows the IRB Administrator to revert the protocol to its state prior to the last protocol action taken. The Undo Last Action works only on the current version of the protocol. It deletes any correspondence generated as a result of the protocol action. It also removes any entries made to the action history of the protocol. The protocol status and submission status also change to the state prior to protocol action being taken.

The Undo Last Action cannot be performed if the preceding protocol action is one of the following: Protocol Created, Submitted to IRB, Renewal Created, Amendment Created, Expired, Withdrawn, Amendment Approval, Renewal Approval and Administrative Correction.

Routing Rule

A record that contains the data for the Rule Attributes specified in a Rule Template. It is an instance of a Rule Template populated to determine the appropriate Routing. The Rule includes the Base Attributes, Required Extension Attributes, Responsible Party Attributes, and any Optional Extension Attributesthat are declared in the Rule Template. Rules are evaluated at certain points in the routing process and, when they fire, can generate action requests to the responsible parties that are defined on them.

COI Disposition Status

The COI Disposition Status maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This status provides information about the project to financial entity relationship within a disclosure. The reporter is presented a limited set of values (No Conflict Exists, Potential Relationships and Relationships Identified) from the COI Disposition Status maintenance table to describe his/her project to financial entity relationship. When an administrator receives the disclosure he is allowed to override the reporter-selected disposition with all the values existing in the COI Disposition Maintenance table. Document Layout.

Note - The system uses the numeric code assigned to each COI Disposition Code to determine which disposition code has the highest value (higher values are considered “bad” statuses, or statuses that require additional review.). The disclosure header panel displays the COI Disposition Statuses with the highest number. For project disclosures with multiple financial entities and for annual and update master disclosures, the disclosure status with the highest value is chosen as the status of the disclosure.

Edit COI Disposition Status Section

The Edit COI Disposition Status section of the COI Disposition Status maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 801 COI Disposition Status Maintenance Document - Edit COI Disposition Status Section Field Descriptions

Field

Description

COI Disposition Code

Required.  An abbreviated form of the Description field (for example, a numeric value) used to uniquely identify the record in the system. 

COI Disclosure Status Code

Required. This field links the COI disposition to the COI Disclosure status code and controls whether the based on the particular COI Disposition the COI Disclosure action of “Approve” or “Disapprove” will be available within the disclosure.

Description

Required.  The long form of the COI Disposition Code field (for example, a common textual name) that describes the purpose of the COI Disposition Code.

Print

Specify all or select portions of this Subaward document to be included in the generated FDP Agreement, and then click the [print] button to create the PDF which you can download/save and then print locally.

Funding Source

If there is more than one Funding Source (Award) linked to the Subaward, the user must select one to provide data for the Agreement/Modification.

Use the drop-down  menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Selections include: all linked Award funding sources.

 

FDP Template

The user must select one of the options by click the radio button beside the item:

FDP Agreement if to generate the initial, full agreement.

FDP Modification to generate the modification.

Select All

Click [select all] to check all the print options; deselect any to remove them from being generated.

Select None

Click [select none] to uncheck any selections.

Print

Click the [Print] button in the farthest right column to generate the FDP Agreement document in a separate browser window.

Contacts

The Contacts section of the Subaward page gives you a way to select a name and type of contact to associate with that person.  After selected, a new subsection appears that displays contact information associated with the person for your convenient reference. 

This information is stored and maintained in what is sometimes named the KC Rolodex.

Tip:  This is inconsistently named in KC – “Non-Employee Lookup” , “Non-Organizational Address Book”, “Address Book”, or “Rolodex” are among the variations depending on the user interface element label and software version.

Contact Type

Required.  This is a list that is prepopulated from the Contact Type ( a.k.a. Sponsor Contact Type) table, for which there is a maintenance document.  For more information, see “Sponsor Contact Type” on page 1522.  Use the drop-down  menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

  Even after the line has been added, this remains a field you can modify without having to delete the line item and add a new one to replace it.  To do so, select a different item from the drop-down list and then click the save button.

Actions

Click the add button to add your entry/selection in the Add: row as a sequentially numbered line item row below.  Click the delete button to remove a previously-added numbered line item row.

For more information about using the standard line item functionality in KC, see “Common Line Item Operations” on page 119 in Overview > Common E-Doc Procedures.

Person Details subsection

Dynamically-displayed from the Rolodex (Non-Employee, Non-Organizational Address Book, Address Book, etc.) record selected, this expandable subsection displays the contact information (address, email, phone, etc.) stored in KC.

Contacts

The Contacts section of the Subaward page gives you a way to select a name and type of contact to associate with that person.  After selected, a new subsection appears that displays contact information associated with the person for your convenient reference. 

This lookup searches the KC Address Book.

Tip:  You may need to create address book documents if you want to list any internal contacts. This panel will not search the KC Person records.

Person or Organization

Required.  Use the lookup tool to search the Address Book. Select a result to populate the field.

Contact Type

Required.  Select a type from the drop-down  menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

This list is still shared with the Award module’s Sponsor Contact Type table.  For more information, see “Sponsor Contact Type” on page 1431.

  Even after the line has been added, this remains a field you can modify without having to delete the line item and add a new one to replace it.  To do so, select a different item from the drop-down list and then click the save button.

Actions

Click the add  button to add your entry/selection in the Add: row as a sequentially numbered line item row below.  Click the delete button to remove a previously-added numbered line item row.

The orange “splat” signifies that this item still needs the user to click the Add button.

Person Details direct inquiry

To review a maintained contacts address details, click on the direct inquiry icon  beside their name to display the address book record detail in a white-box.

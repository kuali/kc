Create Continuation without Amendment

Approved protocols typically expire after a year since approval. If the investigator wants to renew the IACUC approval of a project for 3 years they need to submit a continuation request. The continuation without amendment action allows the investigator to submit a continuation (renewal request for 3 years). The continuation without amendment action allows the investigator to submit a standard continuation request. No changes to the protocol are allowed with this action with the exception of being able to add an attachment.

The system generates a protocol with suffix C00X (where X is 1 for the 1st continuation, 2 for the 2nd continuation and so on).

Click the create button to generate a new Protocol document.  Changes to the data elements and an extension of the expiration date will result upon approval. Upon create action for the continuation without amendment, the system create a continuation protocol with a protocol number derived from (protocol number of the protocol being continued followed by a suffix of C001 for the first continuation, C002 for the second continuation and so on). The protocol sections cannot be updated within a continuation without amendment protocol.

Getting Proposal Budgets Into KC Awards

Q:   Why don’t we include a budget in Institutional Proposal?   

A:    When you use Proposal Development & Budget to generate your Institutional Proposals, you can use Medusa to review and open the proposal. If there are significant budget alterations requested by the sponsor, a copy of that proposal should be created and linked to the same IP number. Alternatively, the budget could be modified in the Award Budget module. When the notice of award is received, the IP record can be selected as a funding source to the award, which then automatically links the development proposal to the award. The development proposal budget is then available to be pulled forward into the Award Budget, where it can be updated as required, following your post award budgeting practices.

 

To get an IP record with a  Development Proposal Budget into a KC Award:

1.

Create an Award and select the Institutional Proposal as the Funding Source.

2.

Finalize the Award or the Time & Money document.

  If you haven’t defined the obligated dates and funds, you’ll get an error when you attempt to create the Award Budget. (Finalizing the T&M document populates the Obligated Funds fields in the Award document)

3.

Create an Award Budget (complete the Name field and execute the New action) and open the version just created.

  When the system creates the version, you’ll notice that there is a search icon in the actions window. It is shaped like a Question mark – which can be confused with panel level help…

4.

Execute the search.

  The budget periods from your proposal budget appear – this is the linking that happens through the Institutional Proposal – so it is therefore important that you don’t forget to link the Award to the Institutional Proposal in Step 1. 

5.

Select the appropriate budget period. 

Bringing in the first budget period is helpful if you have to do a rebudgeting exercise.

 

For more information, see in “Add Funding Proposals” on in Award > Funding Proposals.

Subaward Approval Type

Subaward Approval type  Maintenance Document

The Subaward Approval type  maintenance document is used to define a list of items from which to select in the Subaward e-doc’s Closeout panel.  It allows you to establish and maintain abbreviated codes and full textual name (description) values which are stored in a database table.  This data serves as reference information that is made available for display and selection as options in a list This table is an artifact from the MIT Coeus Subcontracts module and is not directly utilized in the KC Subaward module. The primary key (subAwardApprovalTypeCode) of table SUBAWARD_APPROVAL_TYPE is used as a foreign key in the following tables and has references in their corresponding business objects:

•      table: AWARD_SUBAWARD_TERMS business object: AwardSubAwardTerms

•      table: TEMPLATE_SUBAWARD_TERMS business object: TemplateSubAwardTerms

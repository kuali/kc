Closeout Type

Subaward Closeout  Maintenance Document

The Subaward Closeout  maintenance document is used to define a list of items from which to select in the Subaward e-doc’s Closeout panel.  It allows you to establish and maintain abbreviated codes and full textual name (description) values which are stored in a database table.  This data serves as reference information that is made available for display and selection as options in a list.

Edit Subaward Closeout Type Section

The Edit Subaward Closeout Type section of the maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 902 Closeout Type Maintenance Document - Edit Closeout Type Section Field Descriptions

Field

Description

Closeout Type

Required.  The numeric representation of the content of the Description field, this is the value associated with the status name.  Click within the text box (or press the tab key from a previous field) to reposition the cursor so that it is within the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.

Description

Required.  The textual name of the closeout type as it is commonly referred to which describes the numeric code and summarizes the purpose of the status.  Click the expand text  icon to display a pop-up window with an expanded text entry area if you want more screen real estate to type in, and then click the continue button to close the window and return.  After text has been entered and saved, click the green arrow icon to read it in its entirety in the larger pop-up window, and then click the close button to close the window and return.

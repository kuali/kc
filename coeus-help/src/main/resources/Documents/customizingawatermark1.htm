Customizing a watermark:

The Kuali Coeus application allows the watermark to be customized using the attributes such as watermark verbiage, position, alignment, size and color of the watermark. For each protocol status, a text or an image watermark can be chosen. The Watermark Text is configurable. It allows sixty characters and can have free form text, protocol specific data or both. 

 

If an implementing institution wants to have protocol data appear within the watermark, the protocol data field name must be enclosed within parenthesis. For e.g. to have the protocol approval date appear with a verbiage of  “Active - Approved on”, the Watermark Text for the PROTOCOL STATUS of  “Active” (Status Code = 200) should be  “Active - Approved on: {PROTOCOL_INITIAL_APPROVAL_DATE}.

 

Once the protocol is approved, the PROTOCOL_INITIAL_APPROVAL_DATE is appended to the “Active - Approved on:” text and appears on the attachment as shown below.

 There are only select protocol data fields (known as replacement parameters) such as PROTOCOL_INITIAL_APPROVAL_DATE that are allowed to be included within the watermark text.

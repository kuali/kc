Unit Details

The Unit Details are populated from the Reporter’s Person record. Existing units may be added, deleted, and the designated lead unit modified for use with the Financial Entities, but changes here will not impact or update the Person record. One unit must be designated as the Reporter’s Lead Unit by clicking one box in the Lead column.

Table 118 COI Annual Disclosures > Disclosure Page > Reporter Section > Unit Details Subsection – Field Descriptions

Field

Description

Add (row)

Use the Add: row to insert a unit to the Unit Details table. Locate a unit via the Unit Name search tool or manually enter the exact Unit Number in that field, and then click the Add button.

 

Unit Name (Column)

In the “Add” row, this field contains the lookup icon  to use a search tool to locate the unit to add to this table. Once a unit is identified in the search and returned to the table, the Unit Name field will display the name associated with the number. You will need to click the Add button to complete this task.

 

In other rows, this field is displays the name of the unit, as maintained in the system hierarchy.

Unit Number (column)

*At least one Unit is required. In the “Add” row, the field will be populated automatically with the results of a Unit Name Lookup. Alternately, the Unit Number may be manually typed into the field; if there is an exact match, the unit name will also populate in the row. The Unit Number may consist of letters, numbers, or both. 

In other rows, this field will display the unit number associated with the unit name.

Lead

(Column)

*One Unit must be designated as the Lead.  If only one unit is maintained, it will be the lead by default. When adding units, you may click the Lead box in the Add row to designate the unit you are adding as the primary unit.  The checkbox becomes a radio button after being added as a numbered line item.  To change which unit is the lead, click the radio button in the appropriate unit row.

Actions (column)

Add

In the “Add” row, click the Add button when the Unit Name and Number fields are located to complete this task.

 

Actions (column)

Delete

Use the Delete button on Unit row to remove that unit from the Unit Details table.

You will not be prompted to confirm the deletion. Use the Restore action button at the bottom of the screen to return previously submitted data.

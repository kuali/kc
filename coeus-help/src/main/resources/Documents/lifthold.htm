Lift Hold

 

This action allows the IACUC Administrator to lift the hold on a protocol that is in Active – On Hold status based on a request to lift the hold from the investigators.

 

Table 439 Protocol Document, Protocol Actions Page, Request an Action Section, Available Actions – Lift Hold – Action Attributes

  Action attributes

Description

Who can perform action

IACUC Administrators are allowed to perform this action.

Protocol state prior to action

Prior to the action being performed, the protocol must be in the following state:

 

The protocol status can be Active – On Hold

The submission status is in pending status

 

Protocol state after action

After the action is performed

 

The protocol status changes to Active

The submission status changes to Lift Hold

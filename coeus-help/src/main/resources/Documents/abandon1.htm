Abandon

 

The abandon action allows the investigator to cancel a protocol that has been returned to them for major or minor revisions by the IACUC office.  The abandon action on a protocol serves to make the protocol inactive. No further action can be taken on an abandoned protocol.

Table 405 Protocol Document, Protocol Actions Page, Request an Action Section, Available Actions – Abandon Action - Action Attributes

  Action attributes

Description

Who can perform action

Investigators can initiate this action.

Protocol state prior to action

Prior to the action being performed, the protocol must be in the following state:

 

The protocol status must be in one of the following statuses Minor Revisions Required, Major Revisions Required

The submission status must be in one of the following statuses Minor Revisions Required,  Major Revisions Required

 

Protocol state after action

After the action is performed

 

The protocol status changes to Abandoned

The submission status does not change

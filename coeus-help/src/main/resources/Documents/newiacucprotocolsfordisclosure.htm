New IACUC Protocols for Disclosure

 

Figure 635 New IACUC Protocol for Disclosure Sub-panel

 

This section will list the IACUC Protocols to be disclosed. Expand the IACUC Protocols sub-panel by clicking the [>show]  toggle button.

 

Maintenance note:  The IACUC Protocols in the list are determined by the Status codes maintained in the Config Parameter IACUC_DISCLOSE_STATUS_CODES

 

Figure 636 New IACUC Protocol for Disclosure – expanded to show detail list

 

The New IACUC Protocols for disclosure sub-panel displays the reporter’s IACUC Protocol List.  The identifying information includes Protocol Number, Protocol Name, Protocol Type, and Action.

 

Table 168 New IACUC Protocol for Disclosure field definitions for IACUC Protocol List

Field

Description

(IACUC) Protocol Number

The system-generated number assigned to the protocol document when it was created.

(IACUC) Protocol Name

The title of the proposal as entered by the Aggregator.

(IACUC) Protocol Type

The application type selected for this protocol.

Action

Click on “Report Coi” to begin the New IACUC Protocol (event) Disclosure.

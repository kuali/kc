To initiate a Subaward document:

 

1.  Click the lookup icon for the Site Investigator field in the Subaward section to search for, select and return the desired address book person

2.  Click the lookup icon for the Subrecipient field in the Subaward section to search for, select and return the desired Organization.

3.  Use the Calendar tool to enter a Start Date.

4.  Use the Calendar tool to enter an End Date.

5.  Select an option from the Subaward Type field in the Subaward section (for example, Contract).

6.  Type the identification number in the Purchase Order Id field.

7.  Enter a Title in the text entry field.

8.  Select an option from the Status field (for example, Active).

9.  Click the lookup icon for the Requisitioner User Name, select and return the desired KC person.

10. The Requisitioner’s Unit number will populate to that field. If a different unit should be used, use lookup icon for the Requisitioner Unit number, select and return the value to the field.

11. In the Funding Source section, use the lookup icon to select a funding source from the Award Number field. Click the [add] button to save the selection. The remaining section fields are automatically populated with associated data.

12. In the Contacts section, use the Rolodex Name lookup to search and return a value from the Non-Organizational Address Book.

13. Select a Project Role for the selected Contact, and then click the [add] button to save the selection to the Contacts.

14. Click the savebutton at the bottom, center of the page.
A confirmation message appears in the notification area at the top, left of the page to indicate success.  The Document ID: Status field in the header area displays a unique number and SAVED as the status.

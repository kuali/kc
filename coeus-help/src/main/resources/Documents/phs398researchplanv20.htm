14

Appendix

Attachments

Proposal Tab

Select Proposal Narrative Type ID 34 PHS_ResearchPlan_Appendix

This narrative type requires a Description/Title. Do NOT use special characters or spaces in name you type in KC. Only numbers, letters, hyphens, underscores, and periods should be used.

Please consult the sponsor instructions for your specific opportunity regarding the content requirements for this narrative.

A maximum of 10 PDF attachments is allowed. Note this is the total number of allowable appendix attachments, not the total number of publications. If more than 10 appendix attachments are needed, combine the remaining information into attachment #10. A summary sheet listing all of the items included in the appendix is encouraged, but not required. When including a summary sheet, it should be included in the first appendix attachment.

 

 

Changes from the previous Version 1.3: Removed the pre-filled application type from the form header and reduced the number of narratives. The Enrollment Reports have been converted to separate forms.

Prior form Versions:
Version 1-3 required narrative uploads of the Targeted/Planned Enrollment Report and the Cumulative Enrollment Report to this form. When a prior form version is selected, the appropriate narrative types are made available for selection.

Versions 1-0, 1-1, and 1-2 required the narrative upload types Background and Significance, Preliminary Studies/Progress Report, and Research Design and Methods; replaced by one upload; Research Strategy.

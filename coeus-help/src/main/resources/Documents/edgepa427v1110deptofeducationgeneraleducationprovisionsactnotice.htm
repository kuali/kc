Navigate to the KC Proposal Attachments Screen > Proposal tab

Select the +Add button to create a new attachment

Select the Attachment Type: ED_CertificationDebarment (ID 58)

Select Status: Complete

Note about Unique File Names. KC uses the File Name as the identifier, so to avoid errors at Grants.gov or your sponsor, follow their instructions regarding the proper file name and format. If no instructions are provided, use the following guidance:

Ø The file name of your document should be unique to each upload.

Ø The file name should not contain ANY spaces or special characters, as other characters or symbols will cause errors in transmission.

Use only Numbers 0-9

Uppercase  A-Z or lowercase a-z,

Periods (.), dashes (-), and underscores (_)

In the File section:

Click [Browse]

Ø Search your computer files and select to save your narrative document to this narrative type.

Ø Open/Save the file.

Ø Click the [Save] button to add a row to the Proposal Attachments list.

 

Note that sponsors may request a non-PDF file type for this attachment. The file will be transmitted, but only PDF files are able to be previewed with the Grants.gov S2S Form.

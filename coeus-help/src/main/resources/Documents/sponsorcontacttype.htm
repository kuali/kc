Sponsor Contact Type

 

Lookup Screen:  Contact Type Lookup

Maintenance Document Label:  Contact Type

Channel Title:  Sponsor Contact Type

Edit Section Label:  Edit Contact Type

 

The Sponsor Contact Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.  Contact Type Codes identify points of contact for different areas of responsibility on an award.  For example, Contact Type Codes provide a way to distinguish between administrative, technical, financial, and reporting contacts on a single award. On the Contacts page of the KC Award e-doc, you select a Contact Type from the Project Role list from the Unit Contacts, Sponsor Contacts and Central Administration Contacts subsections in the Unit Contacts section.

Edit Contact Type Section

The Edit Contact Type section of the Sponsor Contact Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 710 Sponsor Contact Type Maintenance Document - Edit Contact Type Section Field Descriptions

Field

Description

Contact Type Code

Required.  Enter a value to identify this Contact Type.  You may use one or more letters and/or numbers.

Description

Required.  Enter a descriptive name for this Contact Type.  For additional fields, consider using a dash followed by a number after the name (for example, Property Office Contact-2).

Modification to existing values are reflected in the corresponding templates that are already defined.

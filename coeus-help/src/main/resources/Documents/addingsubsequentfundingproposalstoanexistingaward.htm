Adding Subsequent Funding Proposals to an Existing Award:

The steps for searching and selecting a subsequent funding proposal is identical to the initial (use the spyglass tool to open the Institutional Proposal Search, enter the search criteria, select a result and return the value to the Award record. The difference between the initial and subsequent funding proposals is in selection how and if personnel data from the source IP should be reflected in the award.

When the award administrator edits an existing award record to add subsequent funding proposals, the Proposal Merge Type options are made available. The administrator has three (3) options to choose how the selected IP record’s KEY PERSONNEL and CREDIT SPLIT information should interact with the Award record: No Change, Merge, or Replace.

Figure 1286 KC Award, Add Funding Proposal Section with Proposal Merge Type options available for selection.

When Key Personnel & Credit Split Data Already Exists In The Award:  You are presented with the following three options:

No change:  Existing award data remains unchanged; no IP information is added to the Award. (The Administrator will review and make any changes manually.)

Merge:  KC combines Funding Proposal Key Personnel & Credit Split data with existing award data. (This usually requires the administrator to review & adjust the credit split information).

Replace:  KC overwrites (replaces) existing Award Key Personnel & Credit Split data with data from the Funding Proposal. (The Administrator will review and make any changes manually.)

 

 

When you select the merge option, these are the resulting implications:

•      KEY PERSONNEL

o  Principal Investigator– Merging an additional Funding Proposal does NOT change the existing Award PI.  Additional PIs are added as Co-Is.

o  Other Key Personnel– Funding Proposal Co-I’s and Key Persons are added without duplication to existing Award Key Personnel.

•      PERSON DETAILS

o  Faculty– Adding a Funding Proposal does NOT change the existing state of the Faculty field for existing key personnel.

o  Effort Fields (Total Effort, Academic Year Effort, Summer Effort, Calendar Year Effort)– Merging a new Funding Proposal with an Award does NOT change the values in corresponding effort fields.

•      UNIT DETAILS

o  Lead Unit– Merging an additional Funding Proposal does NOT change the existing Lead Unit.  Additional Lead Units are added without duplication as non-lead units.
 

o  Unit– Funding Proposal Units are added without duplication to existing Units for an investigator.
 

o  COMBINED CREDIT SPLIT– Merging a new Funding Proposal with an Award adds the values in corresponding credit split fields together if a Unit is duplicated.

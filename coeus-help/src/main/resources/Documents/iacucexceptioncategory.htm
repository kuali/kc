IACUC Exception Category

The IACUC Exception Category maintenance document is used to add or modify a list of exceptions (to the standard procedures performed on animals during research). Implementing institutions maintain a list of possible exceptions or deviations from standard procedures within the IACUC Exception Category reference table.

This data serves as reference information that is made available for display and selection in the Protocol Exceptions page within the IACUC protocol.

Edit IACUC Exception Category maintenance

The Edit action allowed on value (such as Food/Water Restriction) within the Exception Category maintenance document allows you to maintain the each exception category and modify as needed.

 

Table 918  Edit IACUC Exception Category maintenance

Field

Description

Exception Category Code

Required.  Enter a numeric value that uniquely identifies the IACUC Exception Category table record in the system.  Click within the text box to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter the desired value in the box as necessary.

Exception Category

Required.  Enter the full textual name of the IACUC Protocol Exception as it is commonly referred to (see examples below).  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information. 

Active

Required. Check the box to make the protocol exception active and available for selection within the Protocol Exceptions page of an IACUC protocol. Uncheck the box if you wish to inactivate protocol exception and prevent its selection from within Protocol Exceptions page of an IACUC protocol.

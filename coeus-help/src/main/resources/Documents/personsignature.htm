Person Signature

The Person Signature functionality allows institutes to apply the logged-in user’s signature on correspondences generated by the application, when the logged-in user performs the action triggering the correspondence. This functionality was initially designed for IRB/IACUC protocol correspondences, but might be used in future for other applications as well. 

Typically administrators who perform administrative functions would use this feature . For e.g. an IRB administrator can choose to upload an image of their signature to the Person Signature Maintenance table. When the IRB administrator performs an IRB action that triggers a correspondence the IRB Administrator’s signature would be applied to the correspondence upon generation of the correspondence. Schools are also provided the choice to always apply a default signature on correspondence (via the Person Signature Modules Maintenance Table).

Both Person Signature and Person Signature Modules Maintenance table entries are required for the functionality(Person Signature) to work.

Note – The out of the Box Kuali Coeus IRB correspondence templates have a styling element to include the person signature if one is defined.

Create New and Edit Person Signature

The Create New action allowed on Person Signature allows you to select a person and add their autograph to the Person Signature Maintenance table. The Edit action allows you to maintain the person signature record going forward.

 

 

 

Table 753 Edit Person Signature Maintenance Document

Field

Description

KC Person

Use the lookup icon  to select the person whose signature needs to be uploaded into the Person Signature table OR directly enter into the KC Person field.

Autograph

Click the Browse…button to look for, select, and upload the desired image of the signature you want to use.  The field displays the path and filename you selected and the attachment is uploaded upon save.

Active

Required. Indicate whether the Person Signature should be applied or not by using the Active indicator. Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

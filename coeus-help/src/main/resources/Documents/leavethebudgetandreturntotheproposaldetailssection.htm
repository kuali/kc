Leave the budget and Return to the Proposal - Details section

To close the budget and return to the proposal, use the blue [return to proposal] at the top right of the page. This navigation tool appears in every budget tab, at the top of every budget page.

Figure 221 Budget Document > Return to Proposal

 

  Do NOT use the [close] button at the bottom of the page if you want to return to the proposal details section. If you click the [close] button on the bottom of any budget page you will close the entire proposal and be returned to the Researcher, Unit, or Central Admin main navigation tabs from which you started the proposal.

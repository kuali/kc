Add a Performance Site

o  Click on the Add Performance Site lookup button to open the Address Book lookup window

o  Enter your search criteria in the provided fields and then click the Search button at the bottom of the window.

o  Click in the radio button beside the site in the Results window and the scroll down and click the Return Selected button.

•      Or click the Refine Search button to return to the query screen

§ If a new address book entry is required, use the Add A New Address Book button in the upper-right of the search window, or contact your local support desk for assistance.

•      Or click Close.

o  If Return Selected was clicked, the address book record is added as a Performance site.

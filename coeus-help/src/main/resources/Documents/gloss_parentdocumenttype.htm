Parent Document Type

A workflow Document Type from which another Document Type can be derived. The child type  inherits the properties of the parent type, any of which it may override. A Parent Document Type may also have a parent as part of a hierarchy of Document Types.  Parent Document Type is a Document Type from which another Document Type derives from. The child type can inherit certain properties of the parent type, any of which it may override. A Parent Document Type may have a parent as part of a hierarchy of document types .

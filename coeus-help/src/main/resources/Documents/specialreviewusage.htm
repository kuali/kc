Special Review Usage

The Special Review Usage maintenance document is used to establish and maintain numeric code values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.  It assigns an ID code to a Special Review Type Code that is associated with a particular Module Code.

A special review usage is essentially a combination of Speicial Review Type Code (for example, Animal Usage) and a Module Code (for example, Award).  Other areas of the system can then refer to this combination by a single identification number.

Edit Special Review Usage Codes Section

The Edit Special Review Usage Codes section of the Special Review Usage maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

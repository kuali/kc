Institutional Proposal

The Institutional Proposal page contains the key data for a proposal and its awards.  It is the first page when you open an Institutional Proposal document.  It contains these sections: Document Overview, Institutional Proposal, Sponsor & Program Information, Financial, Graduate Students, Notes and Attachments (with count) Delivery Info, and Keywords. See the table below for descriptions of the content in each panel section.

 

When you first open the IP document from the search window, it will be in READ ONLY mode. Click the (edit) button located at the bottom of the screen to make any additions or changes to the document.

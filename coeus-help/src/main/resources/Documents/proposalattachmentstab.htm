Proposal Attachments Tab:

This is the screen where narrative attachments are uploaded for inclusion in your proposal submission. The Attachment Types available from the drop-down list will depend on the type of submission. If an S2S opportunity has been linked to the proposal, the list of Attachment Types will include those types that ‘map’ to the electronic form. Please see Appendix B Grants.gov S2S Forms Guide for detailed, form-by-form information regarding attachment types for s2s forms. If no S2S opportunity is linked, a shorter list of attachment types is displayed. Any type of file may be uploaded, saved, and transmitted; but please refer to sponsor instructions for the specific file type required.

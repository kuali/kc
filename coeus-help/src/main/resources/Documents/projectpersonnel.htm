Project Personnel

 

The Project Personnel tab always displays ‘(All Periods)’ in the tab label.  Changes made to this tab must be saved prior to adding details to the budget.  For deletions, you must delete from the details first, then confirm the delete action on the Project Personnel tab to complete the delete process.

Actions

 

Click to remove an individual.  Note:  you must delete from the details first, then confirm the delete action on the Project Personnel tab to complete the delete process.

Click [Base Salary by Period] to open a pop-up window to maintain the salary values to appear in the Base Salary field in certain S2S budget forms.

You may do any of the following to populate the fields

•      Manually enter the base salary value

•      Click [calculate] to have the system multiply the maintained base salary by the value maintained in the parameter document DEFAULT INFLATION RATE FOR SALARY

•      After using the calculate option , manualy adjust the values.

•      Click [save] to maintain the values

•      [Click [close] to close this window and return to the Personnel budget page.

The system brings forth the information associated with any individuals added to the Key Personnel section after a budget version has been created.  The sync button only adds information; it does not delete any budget person.  (e.g. if a person is deleted from the Proposal Key Personnel section, pressing sync on the Project Personnel section would not delete the corresponding entry on the Project Personnel section.

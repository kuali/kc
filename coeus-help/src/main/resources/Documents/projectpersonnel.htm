Project Personnel

Use the Project Personnel screen to add or delete personnel who will be included in the budgeted.  Any project personnel added in the Key Personnel in the Proposal Details will automatically appear in the Project Personnel list so you do not have to search for them a again. You may not need or wish to add all of these pre-populated persons to your budget detail. This screen is where you add any other (non-key) personnel, as necessary (employees, non-employees, and to-be-named individuals).

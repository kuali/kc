Reporter: Manual Disclosure

The Reporter’s information comes from the Person data associated with the logged in user. If any of the contact details are incorrect, they can be updated in KC by a user with the appropriate roles via the navigation path: System Admin > Identity > Person. Most KC implementations update/maintain this Person data table with a regularly scheduled HR Data feed. The Reporter must have one unit designated as a Lead. Units may be added, deleted, and designated as lead for use with the Financial Entities in that subsection, but changes here will not impact or update the Person record.

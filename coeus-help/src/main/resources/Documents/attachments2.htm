Attachments

Figure 326 COI Annual Disclosures > Disclosure Page > Attachments section entry area

 

To maintain an attachment, select an Attachment Type from the drop-down list.

If desired, you may enter a contact name, email, and phone number in the fields provided.

If the attachment pertains to a listed project, financial entity, or both, you may select one from the drop down list.

Enter a comment in the text field, if desired.

A description of the attachment is required in the Description field.
Use the “Choose File” button to search for and select the document to upload.

Click the Add button to complete this attachment upload.

 

Once added, you can expand the subsection to view, replace or delete the attachment.

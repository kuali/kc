Using the User Documentation

 

The user documentation you are reading is a part of a comprehensive set that delivers three primary output formats for the same source content.

•      PDF:  This documentation set is available in printable PDF format which requires the use of a portable document reader application such as Adobe Acrobat or equivalent.  This format is ideal for printing and electronic navigation.

•      WORD:  It is also available in MS Word (.docx) format so that you may customize it for your institution.  This may be edited with free programs such as Open Office as well.  This format is ideal for printing and editing.

•      HTML:  Lastly, the same content is published as in-application help (HTML) which you may customize using any editor such as Amaya (free) or Dreamweaver.  This format has efficient search capabilities and is optimized for online viewing.

Summary Submit

The Summary Submit screen provides a series of panels of the proposal’s data targeted for the reviewer/approver.  Proposal preparers can review the summary for completion prior to using the Submit for Review button as well as using the View Route Log button to preview the approval routing prior to submission.

Summarized budget information will appear if the parameter "enableBudgetSummaryPanel" is set to “Y.” (not shown).

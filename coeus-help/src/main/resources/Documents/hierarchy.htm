Hierarchy

The Hierarchy toolbar tool allows the user to link a series of proposals that may have been created to isolate different budget rates, tasks, personnel, or other requirements. Hierarchy is used to consolidate the proposals as one proposal hierarchy which is routed and submitted as a single proposal.

User may create a hierarchy by using the tool from a current proposal document; the tool copies all the information in the proposal to make a ‘parent’ proposal, and automatically links the initiating proposal as a ‘child.’ 

User can then

•      Search from the child proposal and link to a parent,

•      Search from a parent proposal and link child proposals.

•      Synchronize all child data from the parent.

•      Synchronize child data from one proposal to the parent

•      Unlink a child proposal from the hierarchy – from the parent or from the child.

Budget Categories

 

The Budget Category maintenance document is used to define categories used in the system to support populating sponsor budget forms. Users can modifiy budget category assignment at the line item level in their proposals, thus it is not necessary to make global changes by editing the maintenance document items for a single sponsor requirement. These documents are mostly used for reference to understand the existing budget categories.

Existing Value Modification:  Existing values can be edited as long as the description is synonmous with the original term. You can add new values. You should not delete existing values.

Edit Budget Category Section

The Edit Budget Category section of the Budget Category maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 722 Budget Category Maintenance Document - Edit Budget Category Section Field Descriptions

Field

Description

Budget Category Code

Required.  The unique value used for identification of this budget category in the system as an abbreviated version of its full name in the Description.

Category Type

A classification of budget category that is an abbreviated version of the complete name.  Use the Lookup or Direct Inquiry icons as desired to search for and select the type.  Type the suspected value in the box and click the direct inquiry icon or click the lookup icon to search for and return the value to populate the box automatically.

Description

Required.  Enter a textual name for this budget category that uniquely represents a summarized explanation of its meaning and/or purpose (for example, a title, several words separated by spaces, or a short phrase).

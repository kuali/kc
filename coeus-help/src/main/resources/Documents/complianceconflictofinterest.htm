Brief Description

COI Attachment Type

Establish and maintain different attachment type selections presented to reporters, reviewers, and COI Administrators within a disclosure.

COI Disclosure Status

Maintenance table for statuses applicable to the entire disclosure.

 COI Disposition Status

Maintenance table for statuses applicable to project to financial entity relationships 

COI Committee Role Type

(Not currently in use as of 5.2.1)

COI Disclosure Event Type

Maintenance table for different types of event, manual, annual and master disclosures

COI Note Type

Establish and maintain different note type selections presented to reporters, reviewers, and COI Administrators within a disclosure.

COI Review Type

(Not currently in use as of 5.2.1)

COI Reviewer

Maintenance table for different types of reviewers.

COI Status

(Not currently in use as of 5.2.1)

Entity Relationship Type

Maintenance table for different types of persons for whom financial conflicts of interest might need to be reported via the financial entity relationship matrix table.

Entity Status

Maintenance table to maintain different statuses of the financial entity (active or inactive)

Entity Data Groups

Maintenance table for different high-level categories of financial conflict of interest such as Ownership Interests, Offices and Positions etc which are displayed within the financial entity relationship matrix table.

Entity Data Matrix

Maintenance table to define the different financial conflict of interest categories and the type of GUI component (such as checkbox, dropdown etc), the lookup argument name that appear within the financial entity relationship matrix table.

COI Review Status

Maintenance table for statuses pertaining to the review of the disclosure. Some are set automatically by the system and some are set manually by the COI Administrator.

COI Recommended Action Type

Maintenance table for recommendations that can be provided by COI reviewers

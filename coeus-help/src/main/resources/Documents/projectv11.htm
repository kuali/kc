Project V1.1

The Grants.gov Grants.gov form Project Narrative Attachment Form V1-1, has the capacity to transmit up to one-hundred (100) individual files.

KC Narrative Type: Proposal_Attachments (Proposal Narrative Type ID 53; allows multiples)

 

Instructions: Select and save a Grants.gov opportunity containing the Project V1-1 form in your federally-sponsored proposal.

Navigate to the KC Proposal Attachments panel of the Abstracts and Attachments tab

Select the Narrative Type: Proposal_Attachments

Browse/return attachment:

Ø Search and select to save your narrative document to this narrative type.

Enter a Description. This is required for this Multiple-type narrative.

Ø The entered title should be unique to this upload.

Ø The text entered into the KC field should not contain ANY spaces or special characters, as other characters or symbols will cause errors in transmission.

Use only Numbers 0-9

Uppercase  A-Z or lowercase a-z,

Periods (.), dashes (-), and underscores (_)

Save the narrative.

Ø Click the ‘add’ button

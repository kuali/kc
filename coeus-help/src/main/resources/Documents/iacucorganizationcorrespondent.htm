IACUC Organization Correspondent

The IACUC Organization Correspondents maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.

The IACUC Organization Correspondents maintenance table contains a list of organization level correspondents. Each person listed in the maintenance table is assigned a specific correspondent type. The correspondents defined in this table receive correspondences/notifications about the protocol, if organization correspondents are designated as recipients to the IACUC notification.

A derived role of KC-IACUC: Organization Correspondent is defined to allow selecting organization correspondents as recipients to IACUC       notifications.

 

Existing Value Modification:  Existing values may be edited as long as the description is synonymous with the original term.  New values can be added.  You should not delete existing values.

Edit IACUC Organization Correspondents Section

The Edit Organization Correspondents section of the Organization Correspondents maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

 

Table 924 IACUC Organization Correspondent Type Maintenance Document - Edit IACUC Organization Correspondent Type Section Field Descriptions

Field

Description

KC Person

Use the lookup icon provided to select the person you want to designate as an organization correspondent or enter directly into the KC Person field.

Organization Id

Use the lookup icon provided to select the organization you want to associate the person selected in the KC Person field or enter directly into the Organization Id field.

Correspondent Type Code

Use the lookup icon to assign a correspondent type to the person selected in the KC Person field or enter the Correspondent Type Code directly into the field. Out of the box, Kuali Coeus provides 3 values “Administrative contact” ,“IACUC correspondent”, “Organization contact”.

 

Non-Employee flag

Check this box to indicate that the person selected in the KC Person field is a non-employee. Uncheck to indicate the person is an employee

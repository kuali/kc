Delete Protocol, Amendment, Renewal, or Continuation

This action allows deletion of a protocol, amendment, renewal, or continuation that has never been submitted into the system. After deletion the protocol is marked inactive by the system. It can still be accessed through protocol lookup by searching for inactive protocols.

 

Table 429 Protocol Document, Protocol Actions Page, Request an Action Section, Available Actions – Delete Protocol, Amendment, Renewal or Continuation – Action Attributes

  Action attributes

Description

Who can perform action

Investigators can perform this action.

Protocol state prior to action

Prior to the action being performed, the protocol must be in the following state:

 

The protocol status must be Pending/In Progress or Amendment in Progress, or Renewal in Progress, or Continuation in Progress.

 

Protocol state after action

After the action is performed

 

The protocol status changes to Deleted

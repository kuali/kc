KR 1.0.3.3

KC and KFS will interact by each exposing services needed by the other system on the Kuali Service Bus, a module within Kuali Rice (KR). The consumer of the service will have an internal interface fronting a default implementation which makes the call over the KSB. This allows implementers to override the default implementation and integrate with non-Kuali applications, if they so choose.

•      New Web Services on the Kuali Service Bus (KSB)

o  BudgetCategoryService allows KFS to look up KC Budget Categories

o  CFDA data querying service allows KC to update CFDA table from cfda.gov website

o  Service to query Sponsor Federal Pass Through

o  Service to query Award Project Director

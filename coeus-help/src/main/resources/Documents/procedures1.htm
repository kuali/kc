Procedures

 

The Procedures section requires the investigator to select the procedures or procedures they want to perform on the species.

Once the investigator selects one or more procedures, will the system allow investigators to enter data related to each procedure in the following tabs running horizontally across within Procedure Details

Ø Procedures

Ø Personnel

Ø Location

The last tab “Summary” serves to consolidate and present all information entered in Procedure, Personnel and Location tabs in a single view.
